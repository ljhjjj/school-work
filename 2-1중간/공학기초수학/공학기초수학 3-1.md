# 3-1. 선형연산자와 냉각방정식의 해석

## 1. 선형성(Linearity)의 정의

수학에서 어떤 대상이 '선형적'이라고 하려면 다음의 두 가지 조건을 반드시 만족해야 합니다.

- **중첩의 원리 (Superposition):** $f(x+y) = f(x) + f(y)$
    
- **비례의 원리 (Homogeneity):** $f(cx) = cf(x)$
    

| **구분**     | **대상 (Input)** | **출력 (Output)** | **예시**      |     |
| ---------- | -------------- | --------------- | ----------- | --- |
| **선형 함수**  | 숫자 ($x$)       | 숫자 ($f(x)$)     | $f(x) = ax$ |     |
| **선형 연산자** | 함수 ($f$)       | 함수 ($L(f)$)     | 미분, 적분 등    |     |

---

## 2. 선형 연산자 (Linear Operator, $L$)

연산자(Operator)는 함수를 입력받아 새로운 함수를 내놓는 '기계'와 같습니다.

### $L = \frac{d}{dt} + k$ 가 선형임을 증명

우리가 1계 선형 ODE에서 자주 보던 이 연산자가 왜 선형인지 확인해 봅시다.

1. **비례 확인:** $L(cf) = (cf)' + k(cf) = c(f' + kf) = cL(f)$ (**만족**)
    
2. **중첩 확인:** $L(f+g) = (f+g)' + k(f+g) = (f' + kf) + (g' + kg) = L(f) + L(g)$ (**만족**)
    

> **결론:** $L = \frac{d}{dt} + k$ 는 선형 연산자입니다.

---

## 3. 선형 연산자의 강력한 성질

연산자가 선형이면, 복잡한 문제를 잘게 쪼개서 풀 수 있습니다.

### (1) 제차 방정식 (Homogeneous Case)

- $Ly_1 = 0$ 이고 $Ly_2 = 0$ 이면, 이들의 합인 $L(y_1 + y_2) = 0$ 도 성립합니다.
    
- 즉, **해들의 선형 결합도 역시 해가 됩니다.**
    

### (2) 비제차 방정식 (Non-homogeneous Case)

- 외력 $f$ 가 $f = f_1 + f_2$ 로 나뉜다면, 각각에 대한 해($Ly_1 = f_1, Ly_2 = f_2$)를 구해 더하기만 하면 전체 해가 됩니다.
    
- **$L(y_1 + y_2) = f_1 + f_2 = f$**

## 4. 냉각 방정식의 응용 및 물리적 해석

냉각 방정식 $\frac{dT}{dt} = k(T_e - T)$ 를 연산자 관점에서 정리하면 $T' + kT = kT_e$ 가 됩니다. 여기서 외력(Ambient Temperature) $T_e(t) = \sin(\omega t)$ 일 때의 해를 분석해 봅시다.

### 적분형 해 (Integral Solution)

$$T(t) = \underbrace{k \int_{0}^{t} e^{-k(t-s)} T_e(s) ds}_{\text{Steady-state (과거의 누적)}} + \underbrace{T(0)e^{-kt}}_{\text{Transient (초기값의 영향)}}$$

### 물리적 통찰 (Physical Insight)

1. **초기값의 소멸 (Transient):** $T(0)e^{-kt}$ 항은 시간이 지날수록($t \to \infty$) 0으로 수렴합니다. 결국 시스템의 초기 상태는 잊혀집니$\Pi$다.
    
2. **기억의 가중치 (Weighting Function):** 적분 기호 안의 $e^{-k(t-s)}$ 부분에 주목해야 합니다.
    
    - $s = t$ (현재 시간)에 가까울수록 값은 1에 가깝습니다.
        
    - $s$ 가 작을수록(먼 과거일수록) 값은 0에 가까워집니다.
        
3. **결론:** 현재의 온도 $T(t)$ 는 **아주 먼 과거의 외력은 잊어버리고, 현재와 가까운 시점의 주변 온도($T_e$)에 가장 큰 영향**을 받습니다.
