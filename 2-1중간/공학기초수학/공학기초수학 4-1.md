# 4-1. 2계 선형 ODE: 복소수 근 (Case 3)

이전 강의([공학기초수학 3-5](공학기초수학/공학기초수학%203-5.md))에서 상수 계수를 갖는 2계 제차 미분방정식 $y'' + ay' + by = 0$의 해를 구하기 위해 특성방정식 $\lambda^2 + a\lambda + b = 0$을 풀었습니다. 이번 시간에는 판별식 $D < 0$이어서 **두 개의 복소수 근**을 가지는 마지막 케이스를 다룹니다.

## 1. 복소수 근의 형태와 기본 해

### 1.1 특성방정식의 해 ($\lambda$)

판별식 $D = a^2 - 4b < 0$일 때, 근의 공식에 의해 해는 다음과 같습니다.

$$\lambda = \frac{-a \pm \sqrt{a^2 - 4b}}{2} = -\frac{a}{2} \pm \frac{\sqrt{-(4b - a^2)}}{2} = -\frac{a}{2} \pm i\sqrt{b - \frac{a^2}{4}}$$

여기서 허수부의 계수를 식을 간단히 하기 위해 $\omega = \sqrt{b - \frac{a^2}{4}}$로 치환합니다.

$$\lambda_{1, 2} = -\frac{a}{2} \pm i\omega$$

### 1.2 지수함수 형태의 기본 해 ($y_1, y_2$)

$y = e^{\lambda t}$ 가정에 따라 두 개의 기본 해를 적어보면 다음과 같습니다.

- $y_1 = e^{(-\frac{a}{2} + i\omega)t} = e^{-\frac{a}{2}t} e^{i\omega t}$
    
- $y_2 = e^{(-\frac{a}{2} - i\omega)t} = e^{-\frac{a}{2}t} e^{-i\omega t}$
    

---

## 2. 오일러 공식을 이용한 전개

복소수 형태의 해를 우리가 해석할 수 있는 삼각함수로 바꾸기 위해 오일러 공식($e^{i\theta} = \cos\theta + i\sin\theta$)을 적용합니다.

- $y_1 = e^{-\frac{a}{2}t} (\cos\omega t + i\sin\omega t)$
    
- $y_2 = e^{-\frac{a}{2}t} (\cos\omega t - i\sin\omega t)$
    

**[문제점 발생]** 일반해는 $y = C_1 y_1 + C_2 y_2$입니다. 하지만 $y_1, y_2$가 여전히 허수 $i$를 포함하는 복소 함수(Complex function)입니다. 우리가 다루는 물리적 시스템(스프링 등)의 결과값은 항상 **실수(Real solution)**여야 하므로, 이 형태를 실수 함수로 변환해야 합니다.

---

## 3. 핵심 정리 (Theorem) 및 실수 해 도출

복소수 해에서 실수 해만 뽑아내기 위해 다음의 중요한 정리를 사용합니다.

> **[정리 (Theorem)]** 복소 함수 $y(t) = u(t) + i v(t)$가 실수 계수를 갖는 **선형 제차 미분방정식(Linear Homogeneous ODE)** $(\star)$  의 해라면, 실수부인 $u(t)$와 허수부인 $v(t)$각각도 원래 미분방정식 $(\star)$의 독립적인 해가 됩니다.

### 3.1 정리의 증명 (Proof)

가정에 의해 $y = u + iv$를 식 $y'' + ay' + by = 0$에 대입하면 등식이 성립해야 합니다.

1. **대입:** $(u+iv)'' + a(u+iv)' + b(u+iv) = 0$
    
2. **미분 전개:** $u'' + iv'' + au' + iav' + bu + ibv = 0$
    
3. **실수부와 허수부 분리:** $(u'' + au' + bu) + i(v'' + av' + bv) = 0$
    
4. **결론:** 복소수가 $0$이 되려면 실수부와 허수부가 모두 $0$이어야 합니다.
    
    - $u'' + au' + bu = 0$ ($\implies$ $u$는 $(*)$의 해이다)
        
    - $v'' + av' + bv = 0$ ($\implies$ $v$는 $(*)$의 해이다)
        

### 3.2 실수 형태의 최종 일반해

 $$ y_1 = \underbrace{e^{-\frac{a}{2}t} \cos\omega t}_{u(t)} + i \underbrace{e^{-\frac{a}{2}t} \sin\omega t}_{v(t)} $$
 
 위에 식에  3.1에서 증명한 [정리 (Theorem)]를 적용합니다.

정리에 따라, 괄호 안의 실수부 $u(t)$와 허수부 $v(t)$ 각각이 훌륭한 해가 됩니다. 따라서 이 둘을 새로운 기초 해로 삼아 일반해를 구성합니다.

$$y(t) = C_1 u(t) + C_2 v(t)$$

$$y(t) = C_1 e^{-\frac{a}{2}t} \cos(\omega t) + C_2 e^{-\frac{a}{2}t} \sin(\omega t)$$

_(이제 완벽한 실수(Real) 형태의 해가 되었습니다!)_
