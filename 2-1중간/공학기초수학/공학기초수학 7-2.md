# 7-1. 불연속 함수와 임펄스의 라플라스 변환

## 1. 단위 계단 함수 (Unit Step Function)

특정 시간 이전에는 0이다가, 특정 시간 이후에 값이 1로 점프하여 유지되는 함수로, 전기 스위치를 켜고 끄는 물리적 상황을 모델링하는 데 유용합니다.

- **정의:** $u(t-a)$ 는 $t < a$ 일 때 0, $t \ge a$ 일 때 1의 값을 가집니다.
    
- **구간 자르기 (Cropping):** 두 개의 계단 함수를 조합하여 특정 구간 $[a, b]$ 에서만 1인 함수 $u_{ab}(t) = u(t-a) - u(t-b)$ 를 만들 수 있습니다. 이를 임의의 함수 $f(t)$ 에 곱하면 원하는 구간의 신호만 잘라낼 수 있습니다.
    

## 2. t-이동 정리 (t-Shifting / Second Shifting Theorem)

라플라스 변환은 $0$부터 $\infty$까지 적분하므로, 시간 축에서 평행 이동한 함수 $f(t-a)$ 를 변환할 때, $t<a$ 구간의 정보 상실을 막기 위해 반드시 단위 계단 함수 $u(t-a)$ 를 곱해주어야 합니다.

- **공식:**
    
    $$L\{u(t-a)f(t-a)\} = e^{-as}F(s)$$
    
- **역변환시 주의점:** $F(s)$ 를 역변환하여 $f(t)$ 를 얻은 후, 이를 $a$ 만큼 평행 이동시키고 반드시 앞에 $u(t-a)$ 를 곱해주어야 정확한 해가 도출됩니다.
    

## 3. 디랙 델타 함수 $\delta(t)$  (Dirac Delta Function)와 임펄스

망치로 치는 것과 같이 아주 짧은 순간에 가해지는 강한 충격(Impulse)을 수학적으로 모델링한 특수 함수입니다.

- **개념적 도출:** 폭이 $h$ 이고 높이가 $\frac{1}{h}$ 인 직사각형 펄스의 면적은 항상 1입니다. 여기서 폭 $h \to 0$ 으로 극한을 보내면, 높이는 무한대가 되지만 전체 면적(충격량)은 여전히 1로 유지되는 **델타 함수** $\delta(t)$ 가 됩니다.
    
- **수학적 성질:**
    
    1. **적분(면적):** $\int_{-\infty}^{\infty} \delta(t) dt = 1$
        
    2. **계단 함수와의 관계:** 단위 계단 함수를 미분하면 델타 함수가 됩니다. $u'(t) = \delta(t)$
        
    3. **라플라스 변환:** $L\{\delta(t-a)\} = e^{-as}$ (특히 $a=0$ 일 때, $L\{\delta(t)\} = 1$)
        

## 4. 예제: 임펄스가 가해진 시스템의 응답

초기 상태가 있는 스프링-질량 시스템에 $t = \frac{\pi}{2}$ 시점에서 충격이 가해졌을 때의 움직임을 구합니다.

- **방정식:** $y'' + y = A\delta(t - \frac{\pi}{2}), \quad y(0)=1, y'(0)=0$
    
- **라플라스 변환 대입:**
    
    $$(s^2 Y(s) - s \cdot 1 - 0) + Y(s) = A e^{-\frac{\pi}{2}s}$$
    
    $$(s^2 + 1)Y(s) = s + A e^{-\frac{\pi}{2}s}$$
    
- **$Y(s)$ 정리 및 역변환:**
    
    $$Y(s) = \frac{s}{s^2+1} + A \frac{e^{-\frac{\pi}{2}s}}{s^2+1}$$
    
    첫 번째 항 역변환: $\cos(t)$ 두 번째 항 역변환 ($t$-shifting 적용): $A \cdot u(t-\frac{\pi}{2}) \sin(t-\frac{\pi}{2})$
    
- **최종 해 (Case Format):** $\sin(t - \frac{\pi}{2}) = -\cos(t)$ 이므로, 구간에 따라 나누어 씁니다.
    
    $$y(t) = \begin{cases} \cos(t) & (0 \le t < \frac{\pi}{2}) \\ \cos(t) - A\cos(t) = (1-A)\cos(t) & (t \ge \frac{\pi}{2}) \end{cases}$$
    

## 5. 전달 함수 (Transfer Function)와 임펄스 응답

일반적인 미분방정식 $y'' + ay' + by = f(t)$ 에서 초기 조건이 모두 0일 때, 라플라스 변환을 취하면 다음과 같이 정리됩니다.

$$(s^2 + as + b)Y(s) = F(s)$$

$$Y(s) = F(s) \cdot \frac{1}{s^2 + as + b} = F(s)G(s)$$

- **전달 함수 $G(s)$:** 입력 $F(s)$ 를 출력 $Y(s)$ 로 변환해주는 역할을 하며, 시스템 고유의 특성을 나타냅니다.
    
- **임펄스 응답 $g(t)$:** 입력으로 단위 임펄스 $f(t) = \delta(t)$ 가 들어오면 $F(s) = 1$ 이 되므로, 시스템의 응답은 $Y(s) = 1 \cdot G(s) = G(s)$ 가 됩니다. 이를 역변환한 $g(t) = L^{-1}\{G(s)\}$ 가 바로 임펄스 응답(Impulse Response)입니다.
    
- **결론:** 시스템의 임펄스 응답 $g(t)$ 를 라플라스 변환한 것이 전달 함수 $G(s)$ 이며 , 이를 알면 임의의 입력 $f(t)$ 에 대한 출력은 합성곱 $y(t) = f(t) * g(t)$ 로 구할 수 있습니다.
