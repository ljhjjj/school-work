
## 4.2 복소수의 기원 & 4.3 직교 형식 (Rectangular Form)

복소수의 일반적인 형태는 다음과 같습니다:

$$C = a + bi$$

| 기호    | 의미                        |
| ----- | ------------------------- |
| $$i$$ | 허수 단위 ($$i = \sqrt{-1}$$) |
| $$a$$ | 실수부 (Real part)           |
| $$b$$ | 허수부 (Imaginary part)      |
| $$C$$ | 복소수 전체                    |

**MATLAB 함수:**
```matlab
C = complex(real, imaginary)
```

---

## 4.4 극형식 (Polar Form)

직교 형식을 크기(magnitude)와 각도(angle)로 표현하는 방식입니다.

$$C = |C| \angle \theta$$

$$|C| = \sqrt{a^2 + b^2}$$

$$\theta = \arctan\left(\frac{b}{a}\right) \times \frac{180°}{\pi} \quad \text{(deg)}$$

**MATLAB 구현 방법 (3가지, 결과 동일):**

```matlab
% Description #1: cart2pol 사용
[theta, r] = cart2pol(a, b)

% Description #2: atan 사용
r     = sqrt(a^2 + b^2)
theta = atan(b/a)

% Description #3: abs & angle 사용
r     = abs(C)
theta = angle(C)
```

---

## 4.5 오일러 급수 (Euler's Series)

### 4.5.1 기본 개념

오일러 공식을 이용하면 복소수를 지수 형태로 표현할 수 있습니다:

$$C = C\cos\theta + iC\sin\theta = Ce^{i\theta}$$

- $$a = C\cos\theta$$
- $$b = C\sin\theta$$

### 복소수 연산 예시

두 복소수 $$C_1 = 10e^{i45°}$$, $$C_2 = 20e^{i30°}$$ 에 대해:

| 연산 | 결과 |
|------|------|
| 곱셈 $$C_1 \cdot C_2$$ | $$200e^{i75°} = 200\angle75°$$ |
| 나눗셈 $$C_1 / C_2$$ | $$0.5e^{i15°} = 0.5\angle15°$$ |

> 💡 **핵심 규칙:** 곱셈 → 크기는 곱, 각도는 합 / 나눗셈 → 크기는 나누고, 각도는 차

---

### 4.5.2 오일러 수치해법 — 초기값 문제 (IVP)

오일러 방법의 핵심 점화식:

$$y_{new} = y_{old} + h \cdot f'(x, y)$$

- $$h$$ : 스텝 크기 (step size)
- $$f'$$ : 현재 점에서의 기울기 (선형 근사)

**MATLAB 구조:**
```matlab
% main.m
% 초기조건, 스텝 설정 후 Euler_series 함수 호출

% Euler_series.m
% y' (first derivative) 정의 및 반복 계산
```

- **수치해 (Numeric Solution):** 오일러 방법으로 반복 계산
- **해석해 (Analytic Solution):** 미분방정식을 직접 풀어 비교 검증

## 4.6 푸리에 급수 (Fourier Series)

**임의의 주기함수**를 사인·코사인 함수의 합으로 표현하는 방법입니다.

주기 $$T$$인 함수 $$f(t)$$의 푸리에 급수:

$$f(t) = \frac{a_0}{2} + \sum_{n=1}^{\infty}\left(a_n \cos\frac{2\pi n t}{T} + b_n \sin\frac{2\pi n t}{T}\right)$$

**항의 수에 따른 근사 정확도:**

| 항의 수 | 근사 정확도 |
|---------|------------|
| 1항 | 매우 거친 근사 |
| 2항 | 점진적 개선 |
| 3항 | 더 정밀해짐 |
| 4항 이상 | 원래 함수에 수렴 |

**MATLAB 예시** ($$\omega = \pi,\ T = 2,\ k = 5$$ 인 구형파):
```matlab
% syms 를 이용한 symbolic 연산
% → 미지 변수를 포함한 수식 계산 가능
```

## 4.7 테일러 급수 (Taylor Series)

임의의 매끄러운 함수를 **다항식으로 근사**하는 방법입니다.

$$f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!}(x-a)^n$$

**MATLAB 함수:**

| 함수 | 설명 |
|------|------|
| `taylor(f, var)` | $$var = 0$$ 기준, 5차까지 (매클로린 급수) |
| `taylor(f, var, a)` | $$var = a$$ 기준, 5차까지 (테일러 급수) |
| `taylor(f, var, a, 'order', n)` | $$var = a$$ 기준, $$n$$차까지 |

---

## 4.8 평형점 (Equilibrium Point)

**평형점**이란 미분방정식의 상수 해(constant solution)로 정의됩니다.

$$\frac{dy}{dt} = 0 \quad \text{and} \quad \frac{dx}{dt} = 0$$

**예제 — 연립 미분방정식:**

$$\frac{dy}{dt} = 4x^2 - xy, \qquad \frac{dx}{dt} = 2y - x^2$$

두 조건을 동시에 만족하는 $$(x_1, y_1),\ (x_2, y_2)$$ 를 MATLAB `solve` 함수로 구합니다:

```matlab
syms x y
eq1 = 4*x^2 - x*y == 0;
eq2 = 2*y - x^2   == 0;
[x_sol, y_sol] = solve([eq1, eq2], [x, y])
```
