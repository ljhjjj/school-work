## 6.2 1차원 최적화 (One-Dimensional Optimization)

최적화 문제란 **목적함수(objective function)를 최소화(또는 최대화)하는 결정변수(decision variable)를 찾는 것**입니다.

### 수학적 정의

$$\min_{x} f(x) \quad \text{subject to} \quad x_{lower} < x < x_{upper}$$

- $$f(x)$$ : 목적함수 (Objective function)
- $$x$$ : 결정변수 (Decision variable)
- $$x_{lower},\ x_{upper}$$ : 탐색 범위의 하한·상한

### MATLAB 구현

```matlab
[x, value] = fminbnd(obj, xlow, xup)
```

| 인자 | 역할 |
|------|------|
| `obj` | 목적함수 |
| `xlow` | 하한 (lower limit) |
| `xup` | 상한 (upper limit) |
| `x` (출력) | 최솟값을 만드는 결정변수 값 |
| `value` (출력) | 최솟값 |

> ⚠️ **주의:** 탐색 범위는 경계값을 **포함하지 않음** (open interval)

## 6.3 다차원 최적화 (Multidimensional Optimization)

결정변수가 **벡터 또는 행렬**인 경우의 최적화입니다.

### 수학적 정의

$$\min_{\mathbf{x}} f(\mathbf{x}) \quad \text{subject to constraints}$$

### MATLAB 구현

```matlab
x = fmincon(obj, x0, A, b, Aeq, beq, lb, ub, nonlcon)
```

| 인자 | 설명 |
|------|------|
| `obj` | 목적함수 |
| `x0` | 초기 조건 (initial condition) |
| `A, b` | 선형 부등식 제약 ($$Ax \leq b$$) |
| `Aeq, beq` | 선형 등식 제약 ($$A_{eq}x = b_{eq}$$) |
| `lb, ub` | 하한·상한 (lower/upper bounds) |
| `nonlcon` | 비선형 제약 함수 |

### Example 6.2 — 제약 조건 구조

```matlab
% 초기 조건
x0 = [-1, 0, 0]

% 제약 종류

% 2) Linear constraints

```

## 6.4 선형 계획법 (Linear Programming Optimization)

**목적함수와 제약 조건이 모두 선형**인 최적화 문제입니다.

### 수학적 정의

$$\min_{\mathbf{x}}\ \mathbf{f}^T\mathbf{x} \quad \text{subject to linear constraints}$$

### MATLAB 구현

```matlab
x = linprog(f, A, b, Aeq, beq, lb, ub)
```

| 인자 | 설명 |
|------|------|
| `f` | 선형 목적함수 계수 벡터 |
| `A, b` | 선형 부등식 제약 |
| `Aeq, beq` | 선형 등식 제약 |
| `lb, ub` | 변수 범위 |
| `x` (출력) | 최솟값을 만드는 결정변수 벡터 |

## 6.5 이차 계획법 (Quadratic Programming Optimization)

**목적함수는 이차(quadratic), 제약 조건은 선형**인 최적화 문제입니다.

### 수학적 정의

$$\min_{\mathbf{x}}\ \frac{1}{2}\mathbf{x}^T H \mathbf{x} + \mathbf{f}^T\mathbf{x} \quad \text{subject to linear constraints}$$

- $$H$$ : **헤시안 행렬 (Hessian matrix)** — 이차 항의 계수

$$H_{i,j} = 2\frac{\partial^2(\text{obj})}{\partial x_i \partial x_j}$$

- $$\mathbf{f}$$ : 선형 항의 계수 벡터

### MATLAB 구현

```matlab
x = quadprog(H, f, A, b, Aeq, beq, lb, ub)
```

### Example 10.4 — 구체적 형태

$$\text{obj}(x) = 0.5 \begin{pmatrix} x_1 & x_2 \end{pmatrix} \begin{pmatrix} 4 & 0.5 \\ 0.5 & 6 \end{pmatrix} \begin{pmatrix} x_1 \\ x_2 \end{pmatrix} + \begin{pmatrix} -4 & 1 \end{pmatrix} \begin{pmatrix} x_1 \\ x_2 \end{pmatrix}$$

$$\underbrace{\begin{pmatrix} 4 & 0.5 \\ 0.5 & 6 \end{pmatrix}}_{H} \quad \underbrace{\begin{pmatrix} -4 \\ 1 \end{pmatrix}}_{\mathbf{f}^T}$$

## 6.7 응용: 리튬이온 배터리 최적화

### 문제 설정

리튬이온 배터리 시스템에서 **모듈 수를 최소화**하는 최적 구성을 찾는 문제입니다.

```
구조: Cell → Module → Pack
```

### 제약 조건 요약

| 조건 | 내용 |
|------|------|
| 방전율 | 항상 **100 kWh** 유지 |
| 출력 전압 | **150 ~ 400 V** |
| 허용 오차 | 방전율 **±1%** 허용 |
| 전압 비례 | 열(column) 방향 모듈 수에 비례 |
| 방전율 비례 | 전체 모듈 수에 비례 |
| 셀 분리 | 모듈 내 셀은 개별 분리 **불가** |

### 최적화 설정

```matlab
% 제약 조건 유형: 비선형 (Nonlinear)
% MATLAB 명령어: fmincon

% 초기 조건
x0 = [4, 14]   % 4행 × 14열 → 20.16 kWh, 168V
```

### 목적함수 및 제약 구조

$$\min_{\mathbf{x}}\ f(\mathbf{x}) = \text{총 모듈 수} \quad (x_1 \times x_2)$$

- $$x_1$$ : 행(row) 방향 모듈 수
- $$x_2$$ : 열(column) 방향 모듈 수

## 📌 최적화 방법 총정리

| 방법 | 목적함수 | 제약 조건 | MATLAB 함수 |
|------|---------|----------|-------------|
| **1차원 최적화** | 임의 | 범위만 | `fminbnd` |

| **선형 계획법** | *선형* | 선형 | `linprog` |
| **이차 계획법** | *이차* | 선형 | `quadprog` |
| **배터리 응용** | 비선형 | 비선형 | `fmincon` |
