## Example 1: 포물선 운동 (Parabolic Trajectory of an Arrow)
> 2차원 자유낙하 — **공기저항 없음 (No Damping)**

### 🎯 문제 목표

- 발사체의 **위치** $$(x, y)$$ 및 **속도** (크기, 방향) 계산
- **지면 도달 시간**, **수평 최대 거리**, **최대 높이** 산출
- 공기저항 무시

---

### Step 1: 수식 정리 (Formulation)

**속도 성분:**

$$V_x = V_0 \cos\theta_0$$

$$V_y = V_0 \sin\theta_0 - gt$$

$$V = \sqrt{V_x^2 + V_y^2}, \qquad \theta = \tan^{-1}\left(\frac{V_y}{V_x}\right)$$

**위치:**

$$x = V_0 t \cos\theta_0$$

$$y = V_0 t \sin\theta_0 - \frac{g}{2}t^2$$

**주요 특성값:**

| 항목 | 수식 |
|------|------|
| 수평 최대 거리 | $$x_{max} = \frac{2V_0^2}{g}\sin\theta_0\cos\theta_0$$ |
| 수평 최대 도달 시간 | $$t_{x_{max}} = \frac{2V_0}{g}\sin\theta_0$$ |
| 최대 높이 | $$y_{max} = \frac{V_0^2}{2g}\sin^2\theta_0$$ |
| 최대 높이 도달 시간 | $$t_{y_{max}} = \frac{V_0}{g}\sin\theta_0$$ |

---

### Step 2: MATLAB 코드

```matlab
% ── 1. 입력 변수 정의 ──────────────────────────────
g   = 9.81;
v0  = input('What is the launch speed in m/s? ');
th0 = input('What is the launch angle in degrees? ');
th0 = pi * th0 / 180;          % 도(degree) → 라디안 변환

% ── 2. 비행 거리 및 시간 계산 ──────────────────────
txmax = (2*v0/g) * sin(th0);
xmax  = txmax * v0 * cos(th0);

% ── 3. 시간 배열 생성 ──────────────────────────────
dt = txmax / 100;
t  = 0 : dt : txmax;

% ── 4. 궤적 계산 ───────────────────────────────────
x = (v0 * cos(th0)) .* t;
y = (v0 * sin(th0)) .* t - (g/2) .* t.^2;

% ── 5. 속도 및 방향 계산 ───────────────────────────
vx = v0 * cos(th0);
vy = v0 * sin(th0) - g .* t;
v  = sqrt(vx.^2 + vy.^2);
th = (180/pi) .* atan2(vy, vx);   % atan2 사용 (4사분면 구분)

% ── 6. 최대 고도 시간 및 위치 계산 ────────────────
tymax = (v0/g) * sin(th0);
xymax = xmax / 2;
ymax  = (v0/2) * tymax * sin(th0);

% ── 7. 결과 출력 및 그래프 ─────────────────────────
disp(['Range = ', num2str(xmax), ' m  /  Duration = ', num2str(txmax), ' s'])
disp(['Max altitude = ', num2str(ymax), ' m  /  Arrival = ', num2str(tymax), ' s'])

plot(x, y, 'k', xmax, y(end), 'o', xmax/2, ymax, 'o')
title(['Projectile flight path, v_0 = ', num2str(v0), ...
       '  \theta_0 = ', num2str(180*th0/pi)])
xlabel('x'), ylabel('y')

figure
plot(v, th, 'r')
title('Projectile speed vs. angle')
xlabel('V'), ylabel('\theta')
```

> 💡 **`atan` vs `atan2` 차이점**
> - `atan(y/x)` : 결과 범위 $$(-90°, +90°)$$ — 2사분면과 3사분면 구분 **불가**
> - `atan2(y, x)` : 결과 범위 $$(-180°, +180°)$$ — **전 사분면 정확히 구분 가능** ✅

---

### Step 3: 결과 분석

**테스트 조건:** $$V_0 = 10\ \text{m/s},\ \theta_0 = 45°$$

| 항목 | 결과 |
|------|------|
| 수평 최대 거리 $$x_{max}$$ | $$\approx 10.19\ \text{m}$$ |
| 비행 시간 $$t_{x_{max}}$$ | $$\approx 1.44\ \text{s}$$ |
| 최대 높이 $$y_{max}$$ | $$\approx 2.55\ \text{m}$$ |

## Example 2: 낙하산 강하 (Parachutist — 1D Free Fall with Damping)
> 1차원 자유낙하 — **공기저항 있음 (With Damping)**

### 🎯 문제 목표

- 공기저항 계수 $$c$$에 따른 **속도 프로파일** 계산
- 해석해(Analytical)와 수치해(Numerical, Euler's Method) 비교

---

### Step 1: 수식 정리 (Formulation)

**운동 방정식 (아래 방향 양수):**

$$F = F_D + F_U = mg - cv$$

$$m\frac{dv}{dt} = mg - cv \quad \Rightarrow \quad \frac{dv}{dt} + \frac{c}{m}v - g = 0$$

**해석해 도출 과정:**

특성함수 $$v(t) = e^{\lambda t}$$ 대입:

$$\lambda + \frac{c}{m} = 0 \quad \Rightarrow \quad \lambda = -\frac{c}{m}$$

일반해:

$$v(t) = a_1 e^{-(c/m)t} + a_2$$

초기조건 $$v(0) = v_0$$ 적용:

$$a_1 = -\frac{gm}{c} + v_0, \qquad a_2 = \frac{gm}{c}$$

**최종 해석해:**

$$\boxed{v(t) = \left(-\frac{gm}{c} + v_0\right)e^{-(c/m)t} + \frac{gm}{c}}$$

> 📌 $$t \to \infty$$ 이면 $$v \to \frac{gm}{c}$$ → **종단 속도 (Terminal Velocity)**

---

### Step 2: MATLAB 코드

```matlab
% 해석해 계산
v(:,i) = (-g*m./c + v0) .* exp(-c./m .* t(i)) + (g*m./c);
```

---

### Step 3: 수치해 — 오일러 방법 (Euler's Method)

미분을 차분으로 근사:

$$\frac{dv}{dt} \approx \frac{v(t_{i+1}) - v(t_i)}{t_{i+1} - t_i}$$

점화식:

$$v(t_{i+1}) = v(t_i) + \left(g - \frac{c}{m}v(t_i)\right)(t_{i+1} - t_i)$$

**수치 계산 예시** ($$g=9.8,\ c=12.5,\ m=68.1,\ \Delta t=2,\ v_0=0$$):

| 시간 $$t$$ | 속도 $$v$$ |
|-----------|-----------|
| $$t=0$$ | $$0\ \text{m/s}$$ |
| $$t=2$$ | $$0 + \left(9.8 - \frac{12.5}{68.1} \times 0\right) \times 2 = 19.6\ \text{m/s}$$ |
| $$t=4$$ | $$19.6 + \left(9.8 - \frac{12.5}{68.1} \times 19.6\right) \times 2 = 32.0\ \text{m/s}$$ |
