## 🔧 전체 구조 한눈에 보기

MATLAB 코드는 크게 **초기화 → 조립 → 풀기 → 시각화** 의 흐름으로 구성됩니다.

---

## 📦 STEP 1 — 재료 & 기하학적 물성 정의

가장 먼저 트러스의 **재료 상수**와 **단면 정보**를 변수로 선언합니다.

```matlab
E = 200e9;       % Young's modulus (Pa) = 200 GPa
A = 0.01;        % Cross-sectional area (m^2)
```

- 모든 10개 요소가 **동일한 E, A 값**을 공유
- 이 값들은 이후 강성 행렬 계산에 반복 사용됨

---

## 📍 STEP 2 — 절점 좌표 정의 (Node Coordinates)

6개 절점의 **X, Y 좌표**를 배열로 정의합니다.

```matlab
% [x좌표, y좌표] — 절점 #1 ~ #6
node_coords = [
    0,  3;   % Node 1
    3,  3;   % Node 2
    6,  3;   % Node 3
    6,  0;   % Node 4
    3,  0;   % Node 5
    0,  0;   % Node 6  ← 원점 기준
];
```

- 절점 #6을 원점 `(0,0)`으로 설정
- 수평 트러스 길이 = **3m**, 대각선 트러스 길이 = **3√2 m**

---

## 🔗 STEP 3 — 요소-절점 연결 관계 정의 (Connectivity)

10개 요소 각각이 **어떤 두 절점을 연결**하는지 정의합니다.

```matlab
% [N1, N2] — 각 요소의 시작/끝 절점 번호
elements = [
    1, 2;   % Element #1
    3, 1;   % Element #2  ← 강의에서 "3번과 1번 절점 연결"
    2, 3;   % Element #3
    ...     % Element #4 ~ #10
];
```

> 💡 **핵심 포인트:** 이 연결 정보가 나중에 전체 강성 행렬을 **어느 위치에 더할지** 결정하는 인덱스 역할을 합니다.

---

## ⚡ STEP 4 — 외력 벡터 정의 (Applied Forces)

절점 #2와 #4에 **-Y 방향으로 100 kN** 적용합니다.

```matlab
% 전체 자유도(DOF) 크기 = 절점 수 × 2 = 6 × 2 = 12
F = zeros(12, 1);

% 절점 #2의 Y방향 → DOF index = 2*2 = 4번째
F(4)  = -100e3;   % Node 2, Y방향

% 절점 #4의 Y방향 → DOF index = 2*4 = 8번째
F(8)  = -100e3;   % Node 4, Y방향
```

- 자유도(DOF) 번호 규칙: 절점 `n`의 X방향 = `2n-1`, Y방향 = `2n`
- 나머지 모든 외력은 **0으로 초기화**

---

## 🔒 STEP 5 — 경계 조건 설정 (Boundary Conditions)

절점 #5, #6은 벽에 **완전 고정(Clamped)** → 변위 = 0

```matlab
% 고정 절점의 DOF 번호 목록
fixed_nodes = [
    2*5-1, 2*5,   % Node 5: U5x, U5y
    2*6-1, 2*6    % Node 6: U6x, U6y
];
% = [9, 10, 11, 12]
```

- 이 4개의 DOF는 이후 계산에서 **제거(Reduction)** 됨
- 자유 DOF(Free DOF)만 남겨서 행렬 크기 축소

---

## 🏗️ STEP 6 — 전체 강성 행렬 조립 (Global K Assembly)

**가장 핵심적인 단계**입니다. 각 요소의 국소 강성 행렬을 전체 행렬에 더합니다.

```matlab
K_global = zeros(12, 12);   % 전체 강성 행렬 초기화 (12×12)

for e = 1:10   % 10개 요소 반복
    n1 = elements(e, 1);   % 시작 절점
    n2 = elements(e, 2);   % 끝 절점
    
    % 두 절점 간 각도(θ) 계산
    dx = node_coords(n2,1) - node_coords(n1,1);
    dy = node_coords(n2,2) - node_coords(n1,2);
    L  = sqrt(dx^2 + dy^2);   % 요소 길이
    
    C = dx / L;   % cos(θ)
    S = dy / L;   % sin(θ)
    
    % 4×4 국소 강성 행렬 (전역 좌표계 기준)
    k_local = (A*E/L) * [
         C^2,  C*S, -C^2, -C*S;
         C*S,  S^2, -C*S, -S^2;
        -C^2, -C*S,  C^2,  C*S;
        -C*S, -S^2,  C*S,  S^2
    ];
    
    % DOF 인덱스 매핑
    dof = [2*n1-1, 2*n1, 2*n2-1, 2*n2];
    
    % 전체 행렬에 조립 (해당 위치에 더하기)
    K_global(dof, dof) = K_global(dof, dof) + k_local;
end
```

### 조립 원리 시각화

```
요소 #1 (절점 1-2):          요소 #2 (절점 3-1):
[1,2,3,4] 위치에 더함    +   [5,6,1,2] 위치에 더함
         ↓
    공유 절점(1,2)에서 값이 중첩되어 합산됨
```

---

## 🧮 STEP 7 — 경계 조건 적용 & 변위 계산

고정 DOF를 제거하고 **자유 DOF에 대해서만** 연립방정식을 풉니다.

```matlab
% 전체 DOF 목록
all_dofs  = 1:12;

% 자유 DOF = 전체 - 고정 DOF
free_dofs = setdiff(all_dofs, fixed_nodes);
% = [1, 2, 3, 4, 5, 6, 7, 8]  (절점 1~4의 DOF)

% 자유 DOF에 해당하는 부분 행렬 추출
K_ff = K_global(free_dofs, free_dofs);   % Kff (자유-자유)
F_f  = F(free_dofs);                     % 자유 DOF의 외력

% 변위 계산: U = K^(-1) * F
U_free = K_ff \ F_f;   % \ 연산자 = inv(K_ff) * F_f
```

> 💡 `\` 연산자는 `inv(K)*F` 보다 **수치적으로 안정적**이고 빠릅니다.

### 결과 변위 벡터 재조립

```matlab
U = zeros(12, 1);
U(free_dofs) = U_free;
% 고정 절점(#5,#6)의 변위는 그대로 0 유지
```

---

## 📊 STEP 8 — 요소별 내력 계산 & 시각화

각 요소의 **축방향 내력(Axial Force)** 을 계산합니다.

```matlab
for e = 1:10
    n1 = elements(e, 1);
    n2 = elements(e, 2);
    
    % 요소 기하 정보 재계산
    dx = node_coords(n2,1) - node_coords(n1,1);
    dy = node_coords(n2,2) - node_coords(n1,2);
    L  = sqrt(dx^2 + dy^2);
    C  = dx/L;   S = dy/L;
    
    % 해당 요소의 전역 변위 추출
    dof = [2*n1-1, 2*n1, 2*n2-1, 2*n2];
    U_e = U(dof);   % 4×1 벡터
    
    % 국소 변위로 변환 후 내력 계산
    % F_axial = (AE/L) * [-C, -S, C, S] * U_e
    F_axial(e) = (A*E/L) * [-C, -S, C, S] * U_e;
    
    % 양수 = 인장(Tension), 음수 = 압축(Compression)
end
```

---

## 💡 전체 흐름 요약

```
물성/좌표 정의
    ↓
연결 관계 정의
    ↓
외력 & 경계 조건 설정
    ↓
[루프] 요소별 4×4 k_local 계산
    → K_global에 조립
    ↓
고정 DOF 제거 → K_ff 추출
    ↓
U = K_ff⁻¹ × F_f  (변위 계산)
    ↓
요소별 내력 계산 & 출력/시각화
```

| **단계** | **핵심 연산** | **결과물** |
|----------|--------------|-----------|
| STEP 1~3 | 변수 초기화 | $$E, A, L, \theta$$ |
| STEP 4~5 | 벡터 설정 | $$F$$ 벡터, 고정 DOF 목록 |
| STEP 6 | 행렬 조립 루프 | $$K_{global}$$ (12×12) |
| STEP 7 | 선형 연립방정식 풀기 | $$U$$ 변위 벡터 |
| STEP 8 | 후처리 | 요소별 축력, 변형 형상 |

이 8단계가 FEM 코드의 **표준 골격**이며, 더 복잡한 구조물도 이 흐름을 그대로 따릅니다.
