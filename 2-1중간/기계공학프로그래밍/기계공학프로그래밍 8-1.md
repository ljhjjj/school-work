
## 🎯 1. 선형 프로그래밍이란?

**목적함수(Objective Function)** 를 최소화/최대화하는 **결정변수(Decision Variables)** 를 찾는 최적화 기법입니다.

### 수식 표현 (벡터 형태)

$$\min_{x} \quad \mathbf{c}^T \mathbf{x}$$

$$\text{subject to} \quad A\mathbf{x} \leq \mathbf{b}, \quad \mathbf{x} \geq 0$$

| **기호** | **의미** |
|----------|----------|
| $$\mathbf{x} = [x_1, x_2, ..., x_n]^T$$ | 결정변수 벡터 |
| $$\mathbf{c} = [c_1, c_2, ..., c_n]^T$$ | 목적함수 가중치 벡터 |
| $$A$$ | 제약조건 계수 행렬 |
| $$\mathbf{b}$$ | 제약조건 우변 벡터 |

## 🔧 2. 표준형 변환 3가지 기법

어떤 선형 프로그래밍 문제든 아래 3가지 변환으로 **표준형**으로 바꿀 수 있습니다.

### ① 최대화 → 최소화 변환

$$\max f(x) \equiv \min -f(x)$$

> 목적함수에 **음수 부호**를 붙이면 최대화 문제가 최소화 문제로 변환

### ② 결정변수가 음수일 경우

$$x_j = y_j - z_j \quad \text{where} \quad y_j \geq 0, \; z_j \geq 0$$

> 하나의 변수를 **두 개의 비음수 변수의 차**로 분리

### ③ 부등호 제약조건 → 등호 변환

$$a_1x_1 + a_2x_2 \leq b \quad \Rightarrow \quad a_1x_1 + a_2x_2 + s = b \quad (s \geq 0)$$

> **슬랙 변수(Slack Variable)** $$s$$를 추가해 등호 조건으로 변환

## 🏭 3. 예제 문제 — 기계 부품 생산 최적화

### 문제 설정

부품 1, 2를 각각 $$x$$, $$y$$개 생산할 때 **주당 최대 이익**을 구하라.

| | **부품 1 (x)** | **부품 2 (y)** | **최대 가용 시간** |
|--|--------------|--------------|----------------|
| 선반 (Lathe) | 10분 | 5분 | 2,500분 |
| 밀링 (Milling) | 4분 | 10분 | 2,000분 |
| 연삭 (Grinding) | 1분 | 1.5분 | 450분 |
| **단위 이익** | **$50** | **$100** | — |

### 목적함수 & 제약조건

$$\max \quad f = 50x + 100y$$

$$\text{s.t.} \begin{cases} 10x + 5y \leq 2500 \\ 4x + 10y \leq 2000 \\ x + 1.5y \leq 450 \\ x \geq 0, \; y \geq 0 \end{cases}$$

### 그래프 풀이 (수동)

```
y
↑
│  ░░░░  ← 제약조건을 만족하는 회색 영역
│  ░░G░  ← 최적점 G
│  ░░░
└──────→ x
```

- 3개의 제약조건 직선을 그려 **가능 영역(Feasible Region)** 확인
- 이익 직선 $$50x + 100y = k$$를 평행 이동시켜 **가능 영역과 접하는 최대 k** 탐색
- **최적해:** $$x = 187.5, \; y = 125.0$$
- **최대 이익:** $$\$21,875$$

## 💻 4. MATLAB 코드 구현

```matlab
% 목적함수 계수 (최대화 → 음수 부호)
f = [-50; -100];   % max(50x + 100y) = min(-50x - 100y)

% 제약조건 계수 행렬 A, 벡터 b (Ax ≤ b)
A = [10,  5;    % 선반 제약
      4, 10;    % 밀링 제약
      1, 1.5];  % 연삭 제약

b = [2500; 2000; 450];

% 하한(lb): x1, x2 ≥ 0
lb = [0; 0];

% 선형 프로그래밍 실행
[x, fval] = linprog(f, A, b, [], [], lb, []);

% 결과 출력
disp('=== 최적 생산량 ===');
disp(x);   % [x1; x2]

disp('=== 최대 이익 ($) ===');
disp(-fval);   % 음수 부호 제거해서 실제 이익 표시
```

### `linprog` 함수 입력 구조

```
[x, fval] = linprog(f, A, b, Aeq, beq, lb, ub)
                    ↑  ↑  ↑   ↑    ↑   ↑   ↑
               목적함수 부등호 등호제약 하한 상한
               계수벡터 제약조건     (없으면 [])
```

| **인자** | **이 문제에서** | **의미** |
|----------|--------------|---------|
| `f` | `[-50; -100]` | 목적함수 계수 (음수 처리) |
| `A` | 3×2 행렬 | 부등호 제약 계수 |
| `b` | `[2500; 2000; 450]` | 제약 우변값 |
| `Aeq`, `beq` | `[]`, `[]` | 등호 제약 없음 |
| `lb` | `[0; 0]` | 비음수 조건 |
| `ub` | `[]` | 상한 없음 |

> 💡 강의에서 강조: `help linprog` 또는 MATLAB 공식 문서로 함수 사용법을 먼저 확인하고, ChatGPT는 **보조 참고용**으로만 사용할 것

## ⚠️ 5. 주의해야 할 예외 케이스

### ① 무한 해 (Infinite Solutions)
이익 직선의 기울기가 제약조건 직선 중 하나와 **완전히 일치**하는 경우

```
→ 선분 CG 위의 모든 점이 최적해
→ 해가 무한히 많음
```

### ② 비유계 해 (Unbounded Solution)
제약조건이 **개방된 영역**을 만드는 경우

```
→ 이익을 무한히 증가시킬 수 있음
→ 제약조건 설정이 잘못된 것 → 문제 재검토 필요
```

> 💡 강의에서 권장: `linprog` 실행 전에 **그래프로 가능 영역을 먼저 시각화**해서 제약조건이 올바른지 확인하는 습관을 들일 것

## 💡 핵심 요약

```
① 문제를 수식으로 정형화
   (목적함수 + 제약조건)
        ↓
② 필요시 표준형으로 변환
   (최대화→최소화, 부등호→등호)
        ↓
③ 그래프로 가능 영역 시각화
        ↓
④ linprog() 함수로 최적해 계산
        ↓
⑤ 결과 검증 및 해석
```

이 선형 프로그래밍 기법은 이후 **프로젝트에서 최적화 문제를 설계할 때** 직접 활용할 수 있는 핵심 도구입니다.
