## 🎯 1강 개요 — 응력(Stress)이란?

응력은 **스칼라도, 벡터도 아닌 2차 텐서(2nd Order Tensor)** 입니다.

| **구분** | **수학적 표현** | **예시** |
|----------|--------------|---------|
| 스칼라 | 숫자 하나 | 질량 (10 kg) |
| 벡터 | 크기 + 방향 | 속도 |
| **텐서** | **행렬 형태** | **응력 ← 오늘 주제** |

### 3D 응력 텐서 (일반형)

$$\boldsymbol{\sigma} = \begin{pmatrix} \sigma_x & \tau_{xy} & \tau_{xz} \\ \tau_{xy} & \sigma_y & \tau_{yz} \\ \tau_{xz} & \tau_{yz} & \sigma_z \end{pmatrix}$$

- 대칭 행렬이므로 **독립 성분은 6개** (3D), **3개** (2D)
- 오늘 강의는 **2D 평면 응력(Plane Stress)** 만 다룸

## 📐 2강 핵심 — 응력 변환 & 주응력

### 응력 부호 규칙

| **종류** | **양(+)** | **음(-)** |
|----------|----------|----------|
| 수직응력 $$\sigma$$ | 인장 (늘어남) | 압축 (줄어듦) |
| 전단응력 $$\tau$$ | 각도 < 90° | 각도 > 90° |

### 좌표 회전 시 응력 변환 공식

$$\sigma_{x'} = \frac{\sigma_x + \sigma_y}{2} + \frac{\sigma_x - \sigma_y}{2}\cos 2\theta + \tau_{xy}\sin 2\theta$$

$$\tau_{x'y'} = -\frac{\sigma_x - \sigma_y}{2}\sin 2\theta + \tau_{xy}\cos 2\theta$$

### 주응력 (Principal Stress)

> 전단응력이 0이 되는 방향 = **수직응력이 최대/최소가 되는 방향**

$$\sigma_{1,2} = \frac{\sigma_x + \sigma_y}{2} \pm \sqrt{\left(\frac{\sigma_x - \sigma_y}{2}\right)^2 + \tau_{xy}^2}$$

$$\tan 2\theta_p = \frac{2\tau_{xy}}{\sigma_x - \sigma_y}$$

### 최대 전단응력

$$\tau_{max} = \sqrt{\left(\frac{\sigma_x - \sigma_y}{2}\right)^2 + \tau_{xy}^2}$$

> 💡 **설계 핵심:** 최대 주응력/전단응력이 재료의 **항복응력(Yield Stress)** 을 초과하면 파괴 → 모어 원으로 이를 시각적으로 확인

## ⭕ 3강 핵심 — 모어 원(Mohr's Circle) 작도법

### 모어 원의 구성 요소

| **요소** | **계산식** | **의미** |
|----------|-----------|---------|
| X축 | $$\sigma$$ | 수직응력 |
| Y축 (↓ 양수) | $$\tau$$ | 전단응력 (아래 방향이 +) |
| 중심 | $$C = \dfrac{\sigma_x + \sigma_y}{2}$$ | 평균 수직응력 |
| 반지름 | $$R = \sqrt{\left(\dfrac{\sigma_x-\sigma_y}{2}\right)^2 + \tau_{xy}^2}$$ | 최대 전단응력 |

### 손으로 그리는 순서

```
① σ축(가로), -τ축(세로) 설정
② 중심 C = (σ_avg, 0) 표시
③ 현재 상태 점 P = (σ_x, τ_xy) 표시
④ C와 P를 연결 → 반지름 R 결정
⑤ 원 작도
⑥ 원과 σ축의 교점 → σ1, σ2 (주응력)
⑦ 원의 최상/최하점 → τ_max
```

### 예제 풀이

- 주어진 값: $$\sigma_x = -20$$ MPa, $$\sigma_y = +90$$ MPa, $$\tau_{xy} = +60$$ MPa
- 중심: $$C = \dfrac{-20+90}{2} = 35$$ MPa
- 반지름: $$R = \sqrt{55^2 + 60^2} \approx 81.4$$ MPa
- 주응력: $$\sigma_1 = 35 + 81.4 = 116.4$$ MPa, $$\sigma_2 = 35 - 81.4 = -46.4$$ MPa
- **최대 전단응력 = 81.4 MPa** → 항복응력 70 MPa 초과 → **파괴 위험!**

## 💻 4강 핵심 — MATLAB GUI 코드 구현

### 전체 구조

```
입력 다이얼로그 (inputdlg)
        ↓
문자열 → 숫자 변환 (str2double)
        ↓
중심 & 반지름 계산
        ↓
모어 원 & 주요 점 시각화 (plot)
        ↓
결과 출력 (fprintf)
```

---

### STEP 1 — GUI 입력 다이얼로그

```matlab
prompt  = {'Sigma X (MPa)', 'Sigma Y (MPa)', 'Tau XY (MPa)'};
dlgname = 'Input: Present State of Stress';
numlines = 1;
defans  = {'150', '25', '-30'};   % 기본값 (문자열)
options.Resize = 'on';
options.WindowStyle = 'normal';
options.Interpreter = 'tex';

answer = inputdlg(prompt, dlgname, numlines, defans, options);
```

> 💡 `inputdlg` 반환값은 **문자열(string)** → 반드시 숫자로 변환 필요

---

### STEP 2 — 문자열 → 숫자 변환 & 기본 계산

```matlab
% 문자열 → 숫자 변환
sx  = str2double(answer{1});   % σ_x
sy  = str2double(answer{2});   % σ_y
txy = str2double(answer{3});   % τ_xy

% 중심 & 반지름
C = (sx + sy) / 2;
R = sqrt(((sx - sy)/2)^2 + txy^2);

% 주응력
sigma1 = C + R;
sigma2 = C - R;
```

---

### STEP 3 — 모어 원 좌표 계산

```matlab
% θ: 0~360도를 360등분
theta = linspace(0, 2*pi, 360);

% 원 위의 모든 점 (σ, τ)
sigma_circle = C + R * cos(theta);
tau_circle   = R * sin(theta);
```

---

### STEP 4 — 시각화

```matlab
fig1 = figure(1);
ax1  = axes(fig1);

% 모어 원 (파란색)
plot(ax1, sigma_circle, tau_circle, 'b-', 'LineWidth', 2);
hold on;

% Y축 방향 반전 (τ 아래 방향이 양수)
set(ax1, 'YDir', 'reverse');

% 중심점 (빨간 원)
plot(ax1, C, 0, 'ro', 'MarkerFaceColor', 'r');
text(C, 0, '  Center');

% 현재 상태 연결선 (σ_x,τ_xy) ~ (σ_y,-τ_xy)
plot(ax1, [sx, sy], [txy, -txy], 'b-', 'LineWidth', 1);

% 현재 상태점 P (초록)
plot(ax1, sx,  txy,  'go', 'MarkerFaceColor', 'g');
text(sx,  txy,  '  (σ_x, τ_xy)');

% 반대편 점 (자홍)
plot(ax1, sy, -txy, 'mo', 'MarkerFaceColor', 'm');
text(sy, -txy, '  (σ_y, -τ_xy)');

% 주응력 표시 (빨간 점선)
plot(ax1, [sigma1, sigma2], [0, 0], 'r--', 'LineWidth', 1);
text(sigma1, 0, sprintf('  σ₁=%.1f', sigma1));
text(sigma2, 0, sprintf('  σ₂=%.1f', sigma2));

% 축 레이블 & 제목
xlabel('Normal Stress σ (MPa)');
ylabel('Shear Stress τ (MPa)');
title("Mohr's Circle");
```

---

### STEP 5 — 결과 출력

```matlab
% 주응력 각도 계산
theta_p = 0.5 * atan2(2*txy, sx - sy);
theta_p_deg = rad2deg(theta_p);

fprintf('=== Mohr''s Circle Results ===\n');
fprintf('Center       : %.2f MPa\n', C);
fprintf('Radius       : %.2f MPa\n', R);
fprintf('σ₁ (max)     : %.2f MPa\n', sigma1);
fprintf('σ₂ (min)     : %.2f MPa\n', sigma2);
fprintf('τ_max        : %.2f MPa\n', R);
fprintf('θ_p          : %.2f deg\n', theta_p_deg);
```

## 💡 핵심 요약

```
응력 상태 입력 (GUI)
        ↓
중심 C, 반지름 R 계산
        ↓
모어 원 시각화
        ↓
σ1, σ2 (주응력) 확인
τ_max (최대 전단응력) 확인
        ↓
항복응력과 비교 → 파괴 여부 판단
```

| **변수** | **의미** | **위치** |
|----------|---------|---------|
| $$C$$ | 원의 중심 | σ축 위 |
| $$R$$ | 원의 반지름 | = $$\tau_{max}$$ |
| $$\sigma_1$$ | 최대 주응력 | 원과 σ축 우측 교점 |
| $$\sigma_2$$ | 최소 주응력 | 원과 σ축 좌측 교점 |
| $$\theta_p$$ | 주축 방향각 | 실제 각도의 **2배**가 원 위 각도 |
