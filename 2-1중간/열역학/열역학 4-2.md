## 1. 준평형 과정과 비열의 정의

닫힌 계에서의 단순 압축성 물질에 대한 준평형 과정(Quasi-equilibrium process)의 열역학 제1법칙 식은 운동/위치 에너지를 무시할 때 다음과 같습니다.

$$\delta Q = dU + \delta W = dU + P dV$$

**비열 (Specific Heat):** 물질 단위 질량의 온도를 1도 올리는 데 필요한 에너지의 양을 의미합니다.

비열은 다른 열역학적 속성들의 함수이므로 그 자체로 하나의 '열역학적 속성'입니다. 따라서 어떤 과정을 거치든 처음과 끝 상태만 같으면 동일한 값을 가지는 **경로 무관(Path-independent**)한 성질을 가집니다.

---

## 2. 정적 비열과 정압 비열

**정적 비열 ($C_v$)** 정적 과정에서는 부피 변화가 없으므로 $dV=0$이 되어 $\delta Q = dU$가 됩니다.

$$C_v \equiv \frac{1}{m}\left(\frac{\delta Q}{\delta T}\right)_v = \frac{1}{m}\left(\frac{dU}{\delta T}\right)_v = \left(\frac{\partial u}{\partial T}\right)_v$$

**정압 비열 ($C_p$)** 정압 과정에서는 압력 변화가 없으므로 $dP=0$이 되어 $\delta Q = dH$가 됩니다 ($H \equiv U+PV$이므로).

$$C_p \equiv \frac{1}{m}\left(\frac{\delta Q}{\delta T}\right)_p = \frac{1}{m}\left(\frac{dH}{\delta T}\right)_p = \left(\frac{\partial h}{\partial T}\right)_p$$

---

## 3. 비압축성 물질 (고체와 액체)

열역학에서는 보통 고체와 액체를 비압축성 물질(Incompressible substances)로 취급하여 밀도나 비체적이 일정하다고 가정합니다. 이 경우 비열은 압력/부피와 무관하게 오직 온도만의 함수가 됩니다.

**내부 에너지 미분:**

$$du = C_v dT = C dT$$

비열이 일정하다고 가정할 때의 전체 변화량: $\Delta u = C \Delta T = C(T_2 - T_1)$

**엔탈피 미분:** 원래 $dh = du + P dv + v dP$ 이지만, 비압축성 물질은 $dv = 0$이므로 다음과 같이 축약됩니다.

$$dh = du + v dP$$

---

## 4. 이상 기체의 내부 에너지와 엔탈피

일반적으로 내부 에너지를 $u=u(T,v)$, 엔탈피를 $h=h(T,P)$로 설정하고 전 미분(Total differential)하면 다음과 같습니다.

$$du = C_v dT + \left(\frac{\partial u}{\partial v}\right)_T dv$$

$$dh = C_p dT + \left(\frac{\partial h}{\partial P}\right)_T dP$$

하지만 **이상 기체(Ideal gas**)에서는 $u$와 $h$, 비열이 모두 오직 온도($T$)만의 함수입니다. 따라서 $\left(\frac{\partial u}{\partial v}\right)_T \equiv 0$, $\left(\frac{\partial h}{\partial P}\right)_T \equiv 0$이 되어 식이 단순화됩니다.

$$du = C_v dT$$

$$dh = C_p dT$$

**이상 기체의 변화량 공식:**

$$\Delta u = u_2 - u_1 = \int_{1}^{2} C_v(T) dT = C_{v,ave} (T_2 - T_1)$$

$$\Delta h = h_2 - h_1 = \int_{1}^{2} C_p(T) dT = C_{p,ave} (T_2 - T_1)$$

**참고:** 이상 기체의 등온 과정(Isothermal)에서는 온도 변화가 없으므로 내부 에너지와 엔탈피 역시 변하지 않고 일정합니다 (Iso-u, Iso-h).

---

## 5. 비열비와 완전 기체 (Perfect Gas)

엔탈피의 정의인 $h = u + Pv$에 이상 기체 상태 방정식 $Pv = RT$를 대입하여 미분하면 $dh = du + d(RT)$가 됩니다.

$$C_p dT = C_v dT + R dT$$

$$C_p = C_v + R$$

이 공식에 의해 기체의 정압 비열은 항상 정적 비열보다 기체상수($R$)만큼 더 큽니다 ($C_p > C_v$).

**비열비 ($\gamma$):** 정압 비열을 정적 비열로 나눈 비율입니다.

$$\gamma = \frac{C_p}{C_v}$$

**완전 기체 (Perfect Gas):** 이상 기체 중에서도 비열($C_v, C_p$)이 상수로 일정한 기체를 완전 기체라고 부릅니다.

---

## 6. 내부 에너지($u$)와 엔탈피($h$) 계산 방법

 $\Delta h$ 및 $\Delta u$를 구하는 4가지 접근법입니다. 정확도가 높은 순서대로 나열되었습니다.

1. 열역학적 성질 표(Table) 사용 (가장 정확): Table A.7이나 A.8에 수록된 온도별 $h_T$ 값을 찾아 직접 빼는 방식입니다 ($h_{T_2} - h_{T_1}$). 
	
2. 근사 공식(Approximate formulas) 직접 적분: Table A.6 등에 있는 온도에 대한 다항식 함수 형태의 비열 $C_{p0}(T)$를 적분식 $\Delta h = \int_{1}^{2} C_{p0}(T) dT$에 넣어 적분합니다. 
	
3. 평균 비열($C_{p,ave}$) 사용: 시작과 끝 온도의 평균 온도에서의 비열 값을 찾아 상수로 취급하여 계산합니다. 
	
4. 상온 비열(Perfect gas 가정) 사용 (가장 오차 큼): 실온(보통 300K)에서의 일정한 비열 값을 가져와 그대로 계산합니다.
