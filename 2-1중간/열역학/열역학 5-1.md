# 열역학 5-1: 열린 계에서의 열역학 제1법칙

## 1. 열린 계 (Open System / Control Volume)의 질량 보존

닫힌 계와 달리 열린 계(검사 체적, Control Volume)는 물질(질량)이 경계를 넘나들 수 있으며, 물질이 이동함에 따라 **에너지도 함께 유출입**하게 됩니다.
![](그림들/열역학/5-1-1%20Open%20System%20&%20Control%20Volume.png)
- **질량 보존의 법칙 (시간 변화율 기준):** 계 내부의 질량 변화율은 들어오는 질량 유량에서 나가는 질량 유량을 뺀 값과 같습니다.
    
    $$\frac{dm_{C.V.}}{dt} = \sum \dot{m}_{in} - \sum \dot{m}_{out}$$
    

### 1-1. 질량 유량 (Mass Flow Rate)과 체적 유량 (Volume Flow Rate)

<img src="../그림들/열역학/5-1-2%20Actual%20&%20Average%20flow.png" align="right" width="200" hspace="30"> 파이프 내 유체의 실제 속도 분포는 포물선 형태(벽면에서 느리고 중심에서 빠름)(a)를 띠지만, 계산의 편의를 위해 단면 전체에서 동일한 속도로 움직인다(b)는 **평균 속도($\bar{V}$**)를 사용합니다.



- **질량 유량 ($\dot{m}$):** 단위 시간당 단면을 통과하는 질량 ($kg/s$).
    
    $$\dot{m} = \rho \bar{V} A$$
    
- **체적 유량 ($\dot{V}$):** 단위 시간당 단면을 통과하는 부피 ($m^3/s$).
    
    $$\dot{V} = \bar{V} A$$
    
- **둘의 관계:** 질량 유량과 체적 유량은 밀도($\rho$) 또는 비체적($v$)을 통해 변환 가능합니다.
    
    $$\dot{m} = \rho \dot{V} = \frac{\dot{V}}{v}$$

## 2. 열린 계의 에너지 보존 (유동 일과 엔탈피 유도)

왜 열린 계 에너지 공식에서 내부 에너지($u$) 대신 엔탈피($h$)가 튀어나오는지 단계별로 정리해 드립니다.

### Step 2-1: 기본적인 에너지 평형 식

질량이 이동하면서 그 질량이 가진 에너지(내부 에너지 + 운동 에너지 + 위치 에너지)도 함께 이동합니다.

![386](../그림들/열역학/5-1-3%20Flow%20Work.png)

- **유동하는 유체의 단위 질량당 에너지 ($e$):** $e = u + ke + pe = u + \frac{V^2}{2} + zg$
    
- 이 에너지를 포함한 초기 에너지 평형 방정식은 다음과 같습니다.
    
    $$\frac{dE}{dt} = \dot{Q} - \dot{W}_{total} + \sum \dot{m}_{in}\left(u + \frac{V^2}{2} + zg\right)_{in} - \sum \dot{m}_{out}\left(u + \frac{V^2}{2} + zg\right)_{out}$$
    

### Step 2-2: 유동 일 (Flow Work, $W_{FW}$)의 존재

열린 계에서는 유체가 경계를 통과해 밀고 들어오거나 밀고 나가기 위해 **가상의 피스톤이 유체를 밀어주는 것과 같은 일(Work**)이 필요합니다. 이를 **유동 일(Flow Work**)이라고 합니다.

- 들어올 때 필요한 일: $-\dot{W}_{in} \approx p_i A_i V_i$
    
- 나갈 때 하는 일: $\dot{W}_{out} \approx p_e A_e V_e$
    
- **순 유동 일 (Net flow work):** $\dot{W}_{FW} = p_e A_e V_e - p_i A_i V_i$
    

따라서, 계가 하는 전체 일($\dot{W}_{total}$)은 우리가 실제로 얻는 축 일(Shaft work, $\dot{W}_s$)과 유체를 밀어내는 유동 일($\dot{W}_{FW}$)의 합입니다.

$$\dot{W}_{total} = \dot{W}_s + \dot{W}_{FW}$$

### Step 2-3: 식 대입 및 엔탈피($h$) 치환 과정 🌟

이제 앞서 구한 유동 일을 Step 2-1의 초기 방정식에 대입하여 식을 정리합니다.

1. **전체 일 분리:** $\dot{W}_{total}$ 자리에 $\dot{W}_s + \dot{W}_{FW}$를 넣습니다.
    
    $$\frac{dE}{dt} = \dot{Q} - \dot{W}_s - \dot{W}_{FW} + \sum \dot{m}_{in}(e)_{in} - \sum \dot{m}_{out}(e)_{out}$$
    
2. **유동 일 식 전개:** $\dot{W}_{FW}$를 $pAV$로 풀어씁니다.
    
    $$\frac{dE}{dt} = \dot{Q} - \dot{W}_s - (p_e A_e V_e - p_i A_i V_i) + \sum \dot{m}_{in}(e)_{in} - \sum \dot{m}_{out}(e)_{out}$$
    
3. **$pAV$를 질량 유량($\dot{m}$) 꼴로 변환:** $\dot{m} = \rho AV = \frac{AV}{v}$ 이므로, $pAV = \dot{m} p v$로 쓸 수 있습니다.
    
    $$\frac{dE}{dt} = \dot{Q} - \dot{W}_s - (\dot{m}_e p_e v_e - \dot{m}_i p_i v_i) + \sum \dot{m}_{in}(e)_{in} - \sum \dot{m}_{out}(e)_{out}$$
    
4. **항 묶어주기:** 유동 일 항($\dot{m}pv$)을 질량이 이동하며 가져오는 에너지 항 $\dot{m}(u+ke+pe)$ 안으로 밀어 넣습니다.
    
    $$\frac{dE}{dt} = \dot{Q} - \dot{W}_s + \sum \dot{m}_{in}(u + \mathbf{pv} + ke + pe)_{in} - \sum \dot{m}_{out}(u + \mathbf{pv} + ke + pe)_{out}$$
    
5. **엔탈피($h$) 치환:** 열역학에서 $h \equiv u + pv$로 정의하므로, 괄호 안의 $u+pv$를 엔탈피 $h$로 묶어줍니다.
    
    $$\frac{dE}{dt} = \dot{Q} - \dot{W}_s + \sum \dot{m}_{in}(h + ke + pe)_{in} - \sum \dot{m}_{out}(h + ke + pe)_{out}$$
    

### 💡 결론 (왜 엔탈피를 쓰는가?)

결과적으로, 유체가 시스템을 들락날락할 때마다 발생하는 귀찮은 **유동 일($pv$)을 내부 에너지($u$)에 미리 더해놓은 값인 엔탈피($h$)를 사용**하면, 에너지 방정식을 훨씬 간결하게 쓸 수 있기 때문입니다.
