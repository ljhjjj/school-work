# Note_creation_guide

## 1. 파일 네이밍 및 저장 경로 규칙
- 생성되는 마크다운 파일의 이름과 저장 위치는 반드시 과목별로 분류하여 엄격하게 지정해야 합니다.
- **파일명 규칙:** `[과목명] [주차]-[차시].md` 형태로 작성하세요. 
  - (예: `고체역학 3-1.md`)
- **저장 경로 규칙:** 루트 디렉토리 하위의 `[과목명]` 폴더 내에 파일을 위치시키세요. 
  - (예: `고체역학/고체역학 3-1.md`, `열역학/열역학 5-2.md`)
- **[중요] 폴더 생성(mkdir) 의무화:** 파일을 작성하고 저장하기 전에, 해당 과목의 폴더가 존재하는지 반드시 확인하세요. 폴더가 없다면 터미널 명령(`mkdir -p "[과목명]"`)을 실행하여 디렉토리를 먼저 안전하게 생성한 뒤에 파일을 저장해야 경로 오류(Directory not found)를 방지할 수 있습니다.

## 2. 본문 생성 규칙
- 분석이 끝난 데이터를 바탕으로 실제 마크다운 문서를 작성할 때 다음 규칙을 따릅니다.
- **AI 행동 지침:**
  1. **디테일 보존:** 명시적인 요약이나 축약 요청이 없는 한, 원본 문서(특히 노트 필기)에 포함된 임의의 세부 내용, 예시, 부연 설명, 도출 과정을 절대 생략하지 말고 문서에 모두 꼼꼼하게 반영하세요.
  2. **긴 문서의 분할 작성 및 대기:** 처리해야 할 결과물의 양이 방대하여 한 번에 작성이 어려울 경우, 임의로 내용을 요약하거나 뭉개지 마세요. 대신 논리적인 단원(H2) 단위로 나누어 파일에 작성한 뒤, `DOC_PLAN.md` 상태를 업데이트하고 **"이어서 다음 파트를 작성할까요?"**라고 사용자에게 물어보고 대기하세요.

## 3. 문서 구조 및 서식 규칙
- 출력되는 마크다운 문서는 다음의 서식과 어투를 엄격하게 준수해야 합니다.
- **AI 행동 지침:**
  1. **메인 제목 (H1)**
     - 양식: `# [과목명] [주차/챕터]: [전체 핵심 주제]`
     - 과목명과 숫자가 포함된 제목은 반드시 대제목(`#`)을 사용할 것. (예: `# 고체역학 3-1: 응력과 변형률`)
  2. **대주제 (H2) 및 구분선**
     - 양식: `## 1. 주제명`, `## 2. 주제명` (숫자 넘버링 필수)
     - 각 대주제의 내용이 끝날 때마다 반드시 수평선(`---`)을 넣어 단원을 시각적으로 분리할 것.
  3. **소주제 (H3)**
     - 양식: `### 1) 소주제명`, `### 2) 소주제명`
     - 대주제 내에서 하위 구분이 필요할 때 사용.
  4. **본문 및 목록 (Bullet points)**
     - 설명은 기본적으로 불릿 포인트(`-`)를 사용하여 간결하게 개조식으로 작성할 것.
     - 핵심 키워드나 장치명, 주요 개념은 반드시 볼드체로 강조하고 콜론을 붙일 것. (예: `- **수직응력 (Normal Stress):** 단면에 수직으로 작용하는...`)
  5. **핵심 내용 강조 (Blockquote)**
     - 과정의 결론이나 꼭 암기해야 할 중요한 가정/핵심 개념은 인용구로 묶어서 시각적으로 돋보이게 할 것.
     - 양식: `> **핵심:** [중요한 내용 설명]`
  6. **수식 (GitHub Flavored LaTeX)**
     - 모든 수학/물리 기호와 수식은 GitHub 마크다운에서 공식 지원하는 표준 LaTeX 문법을 사용하세요. (문장 내 인라인 수식: `$수식$`)
     - **[중요: 블록 수식 렌더링 안전성]:** 마크다운 뷰어(GitHub, Obsidian 등)에서 렌더링이 깨지는 것을 완벽히 방지하기 위해, 별도의 행으로 강조되는 블록 수식은 **반드시 앞뒤로 줄바꿈(Enter)**을 포함하여 독립된 블록으로 작성하세요.
       (잘못된 예: `$$\sigma = \frac{P}{A}$$`)
       (올바른 예:
       $$
       \sigma = \frac{P}{A}
       $$
       )
     - 단, 렌더링 충돌을 일으키기 쉬운 복잡한 매크로(예: `\underbrace`)의 사용은 지양하고, 단순화된 수식 기호로 대체하여 렌더링 안정성을 확보하세요. HTML 수식 태그는 사용하지 마세요.
  7. **어투**
     - '~한다', '~이다' 형태의 간결하고 객관적인 문어체(설명체)를 유지할 것.

# 고체역학 3-1: 응력과 변형률

## 1. 응력(Stress)의 개념

물체에 외부 하중이 작용할 때, 이에 저항하기 위해 물체 내부에 발생하는 단위 면적당 내력을 의미한다.

### 1) 수직응력과 전단응력
- **수직응력 (Normal Stress):** 단면에 수직한 방향으로 작용하는 응력이다. 인장응력(+)과 압축응력(-)으로 구분되며 기호는 $\sigma$를 사용한다.
  $$
  \sigma = \frac{P}{A}
  $$
- **전단응력 (Shear Stress):** 단면에 평행하게 작용하여 물체를 미끄러지게 만드는 응력이며 기호는 $\tau$를 사용한다.
  $$
  \tau = \frac{V}{A}
  $$
> **핵심:** 응력은 단순히 가해진 힘의 크기($P$)가 아니라, 힘이 작용하는 면적($A$)에 대한 비율이므로 재료의 파괴 여부를 결정하는 절대적인 기준이 된다.

---

## 2. 변형률(Strain)의 정의

외력에 의해 물체의 형태나 크기가 변하는 정도를 원형에 대한 비율로 나타낸 무차원 물리량이다.

### 1) 수직 변형률
- **수직 변형률 (Normal Strain):** 원래 길이($L$)에 대한 길이의 변화량($\delta$)의 비를 의미하며 기호는 $\epsilon$을 사용한다.
  $$\epsilon = \frac{\delta}{L}$$
- **특징:** 길이가 늘어나면 양(+), 줄어들면 음(-)의 값을 가진다.
