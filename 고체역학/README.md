# 고체역학

## 과목 소개
기계공학의 핵심 기초 학문으로, 외력에 의해 고체 내부에 발생하는 내력(Internal Force)과 변형(Deformation)의 관계를 학습합니다.

## 학습 진도 (1-2 ~ 6-1)
### 기초 개념과 응력·변형률
- [고체역학 1-2](./고체역학%201-2.md): 고체역학 기초, 힘과 모멘트, 평형 방정식
- [고체역학 2-1](./고체역학%202-1.md): 응력의 정의와 종류, 응력 요소 평형
- [고체역학 3-1](./고체역학%203-1.md): 평균응력, 안전지수
- [고체역학 3-2](./고체역학%203-2.md): 변형률(Strain)

### 재료 거동과 하중 해석
- [고체역학 4-1](./고체역학%204-1.md): 응력-변형률 선도와 재료 거동
- [고체역학 4-2](./고체역학%204-2.md): 푸아송의 비, 전단 응력-변형률
- [고체역학 5-1](./고체역학%205-1.md): 축하중(Axial Load)
- [고체역학 6-1](./고체역학%206-1.md): 축변형과 정정/부정정 구조 해석

## 빠른 확인
- 총 주차 노트: **8개**
- 범위: **1-2 ~ 6-1**
