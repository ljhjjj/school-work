# 고체역학 1-2

## 과목소개
기계공학의 핵심 기초 학문으로, 외부 힘에 의해 고체 내부에 발생하는 내력(Internal Force)과 변형(Deformation)의 상관관계를 탐구합니다. 정적 평형 상태에 있는 물체의 응력(Stress)과 변형률(Strain)을 분석하여 재료의 안정성을 평가하는 능력을 기릅니다.

## 학습진도
- **고체역학 1-2:** 일반물리 개념 복습 및 고체역학 기초 (역학의 분류, 힘과 모멘트, 평형 방정식, 내력의 해석)
- **고체역학 2-1:** 응력(Stress)의 정의와 종류 (수직응력, 전단응력, 응력 요소의 평형과 보완성)
- **고체역학 3-1:** 균일 단면봉, 평균응력(평균 수직응력, 평균 전단응력)과 안전지수
- **고체역학 3-2:** Strain (변형률)

## 메모
- 고체역학은 가속도 $a=0$인 정적 평형 상태를 전제로 함.
- 내력 성분 4가지(수직력, 전단력, 비틀림 모멘트, 굽힘 모멘트)의 물리적 의미를 항상 유의할 것.
- 응력 요소 분석 시 부호 규약(Sign Convention)을 일관되게 유지하는 것이 계산 실수를 줄이는 핵심임.
