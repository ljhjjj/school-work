# 공학기초수학

## 과목 소개
상미분방정식(ODE)을 중심으로 1계/2계 선형 미분방정식, 라플라스 변환, 복소수 기반 해석을 학습하는 과목입니다.

- [공학기초수학 1-3](./공학기초수학%201-3.md): 변수분리형 미분방정식

- [공학기초수학 2-3](./공학기초수학%202-3.md): 오일러 공식과 복소수화
