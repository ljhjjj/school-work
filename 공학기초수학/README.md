# 공학기초수학

## 과목소개
* **교과목명**: 공학기초수학 (Foundations of Engineering Mathematics)
* **학기**: 2026년 1학기
* **소개**: 기계공학 전공 필수 과목들의 기초가 되는 상미분방정식(ODE), 복소해석학, 선형대수학 등을 학습합니다. 
* **목표**: 물리적 시스템(진동, 열전달 등)을 수학적으로 모델링하고, 복소수를 활용한 효율적인 해법을 익힙니다.

---

## 학습 진도

* **1-1.** 상미분방정식(ODE)의 기본 개념 및 방향장(Slope Field)
* **1-2.** 해의 존재와 유일성 정리 (Existence and Uniqueness)
* **1-3.** 변수분리형 미분방정식 (Separable ODEs)
* **2-1.** 완전 미분방정식 (Exact ODEs) 및 적분 인자
* **2-2.** 1계 선형 상미분방정식 (First-order Linear ODEs) 및 베르누이 방정식
* **2-3.** 오일러 공식과 복소수를 활용한 ODE 풀이 (Complexification)
* **3-1.** 선형연산자와 냉각방정식의 해석
* **3-2.** 1계 ODE의 응답 해석: Gain과 Phase Lag
* **3-3.** 자율 미분방정식 (Autonomous ODEs)
* **3-4.** 2계 미분방정식의 도입 (Spring-Mass System)
* **3-5.** 2계 선형 ODE: 상수 계수를 갖는 제차 방정식
* **4-1.** 2계 선형 ODE: 복소수 근 (Case 3)
* **4-2.** 오일러 방정식 (Euler's Equation)
* **4-3.** 2계 선형 ODE의 해의 성질과 론스키안(Wronskian)
* **5-1.** 비제차 선형 미분방정식 (Non-homogeneous Linear ODEs)
* **5-2.** 특수해 구하기 3번째 방법: 복소수화 기법 (Complexification)
