# 2-2. 1계 선형 상미분방정식 (First-order Linear ODEs)

## 1. 정의 및 표준형 (Standard Form)

- **일반형:** $a(x)y' + b(x)y = c(x)$
    
    - **제차 (Homogeneous):** $c(x) = 0$인 경우입니다.
        
    - **비제차 (Non-homogeneous):** $c(x) \neq 0$인 경우입니다.
        
- **선형성(Linearity) 조건:** $y^2, \sqrt{y}, \sin(y')$와 같은 종속변수 $y$ 및 그 도함수에 대한 비선형 항이 없어야 합니다.
    
    - 독립변수 $x$에 대해서는 $x^2, \sin(x)$ 등의 비선형 항이 있어도 선형으로 간주합니다.
        
- **표준형 (Standard Form):** $a(x) \neq 0$일 때 양변을 나누어 다음과 같이 만듭니다.
    
    $$y' + p(x)y = q(x)$$
    

---

## 2. 조건별 풀이 방법

### Case 1: $p, q$가 모두 상수인 경우 ($y' - Ay = B$)

이 형태는 **변수분리형[(Separable ODE**)](공학기초수학%201-3.md)으로 풀 수 있습니다.

1. 식 변형: $\frac{dy}{dx} = Ay + B = A(y + \frac{B}{A})$
    
2. 변수 분리: $\frac{1}{y + B/A} dy = A dx$
    
3. 적분 결과: $\ln|y + \frac{B}{A}| = Ax + c \implies y = Ce^{Ax} - \frac{B}{A}$
    

### Case 2: 하나가 상수인 경우 ($y' - Ay = B(x)$)

양변에 $e^{-Ax}$를 곱하여 해결합니다.

1. 식 변형: $e^{-Ax}y' - Ae^{-Ax}y = e^{-Ax}B(x)$
    
2. 미분 통합: $\frac{d}{dx}(e^{-Ax}y) = e^{-Ax}B(x)$
    
3. 적분 및 정리: $y = e^{Ax} \int e^{-Ax}B(x)dx + Ce^{Ax}$
    

### Case 3: $p, q$가 모두 변수인 일반적인 경우 ($y' + p(x)y = q(x)$)

**적분 인자(Integrating Factor, $u(x)$**) 를 도입합니다.

1. 목표: $u(x)y' + u(x)p(x)y = q(x)u(x)$가 $\frac{d}{dx}(uy)$가 되도록 합니다.
    
2. 적분 인자 도출: $u' = p(x)u$ 조건을 만족해야 하므로, $u = e^{\int p(x)dx}$가 됩니다.
    
3. 결과: $\frac{d}{dx}(uy) = q(x)u(x)$를 적분하여 해를 구합니다.
    

---

## 3. 1계 선형 ODE 풀이 단계 요약

1. **표준형 만들기:** $y' + p(x)y = q(x)$ 형태로 정리합니다.
    
2. **적분 인자 계산:** $u = e^{\int p(x)dx}$를 구합니다.
    
3. **양변에 곱하기:** 표준형의 모든 항에 $u$를 곱합니다.
    
4. **적분 및 해 도출:** $\frac{d}{dx}(uy) = qu$를 적분하여 $y$를 결정합니다.
    

---

## 4. 베르누이 방정식 (Bernoulli Equations)

비선형 방정식이지만 적절한 치환을 통해 **선형 ODE로 변환(Transformation)** 가능한 형태입니다.

- **형태:** $y' + p(x)y = q(x)y^n$
    
- **변환 과정:**
    
    1. 양변을 $y^n$으로 나눔: $y^{-n}y' + p(x)y^{1-n} = q(x)$
        
    2. 변수 치환: $v = y^{1-n}$으로 설정합니다.
        
    3. 미분 관계식: $v' = (1-n)y^{-n}y'$ 이므로 $y^{-n}y' = \frac{v'}{1-n}$입니다.
        
    4. 대입 결과: $\frac{v'}{1-n} + p(x)v = q(x)$
        
- **결론:** 치환 후 정리하면 $v' + (1-n)pv = (1-n)q$ 형태의 **1계 선형 ODE**가 되어 위에서 배운 방식으로 풀 수 있습니다.
