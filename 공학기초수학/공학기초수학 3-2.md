
# 3-2. 1계 ODE의 응답 해석: Gain과 Phase Lag

## 1. 응답 (Response)의 이해

시스템이 외부의 자극(입력)에 대해 시간에 따라 어떻게 반응하는지를 나타내는 결과값(출력)입니다. 미분방정식 $y' + ky = f(t)$에서 우변 $f(t)$가 **입력(Input**)이라면, 해 $y(t)$가 바로 **응답(Output**)이 됩니다.

일반적으로 공학 시스템의 응답은 다음 두 가지의 합으로 구성됩니다.

- **과도 응답 (Transient Response):** 시스템 가동 초기 상태의 적응 과정으로, 시간이 지나면($t \to \infty$) 결국 0으로 사라지는 거동입니다. (제차해 형태)
    
- **정상 상태 응답 (Steady-state Response):** 초기 적응이 끝나고, 입력에 완전히 동기화되어 영구적으로 남는 거동입니다. (특수해 형태)
    

## 2. 복소 미분방정식의 풀이 (Solving the Complex ODE)

외력(Input)이 주기적인 복소수 지수함수($e^{i\omega t}$)인 시스템을 가정합니다.

$$\frac{d\hat{T}}{dt} + k\hat{T} = k e^{i \omega t}$$

### Step 1: 적분 인자(Integrating Factor) 적용

표준 해법에 따라 적분 인자 $u(t) = e^{\int k dt} = e^{kt}$를 양변에 곱합니다.

$$\frac{d\hat{T}}{dt} e^{kt} + k e^{kt} \hat{T} = k e^{kt} e^{i \omega t}$$

$$\frac{d}{dt}(\hat{T} e^{kt}) = k e^{(k+i\omega)t}$$

### Step 2: 양변 적분 및 일반해 도출

양변을 $t$에 대해 적분하면 다음과 같습니다.

$$\hat{T} e^{kt} = k \int e^{(k+i\omega)t} dt = \frac{k}{k+i\omega} e^{(k+i\omega)t} + \hat{C}$$

최종적으로 $\hat{T}$에 대해 정리하면:

$$\hat{T}(t) = \underbrace{\frac{k}{k+i\omega} e^{i \omega t}}_{\text{Steady-state}} + \underbrace{\hat{C} e^{-kt}}_{\text{Transient}}$$

> **💡 물리적 해석 (3-1 복습)**
> 
> $\hat{C} e^{-kt}$ 항은 $t \to \infty$일 때 0으로 수렴합니다. 즉, 초기 상태의 영향은 사라지고(Transient), 결국 입력 주파수 $\omega$에 반응하는 **정상 상태 응답(Steady-state response)**만 남게 됩니다.

## 3. 복소 이득(Complex Gain) 분석

우리가 구하려는 실제 해는 $T(t) = \text{Re}(\hat{T})$입니다. 이를 위해 앞의 계수 $w = \frac{k}{k+i\omega}$를 극형식(Polar form)으로 변환합니다.

### 3.1 계수 $w$의 정리

$$w = \frac{k}{k+i\omega} = \frac{1}{1 + i(\frac{\omega}{k})}$$

이 복소수 $w$를 $w = |w|e^{i\theta}$라고 하면, 크기($|w|$)와 편각($\theta$)은 다음과 같습니다.

- **크기(Magnitude):** $|w| = \frac{1}{\sqrt{1 + (\frac{\omega}{k})^2}}$
    
- **편각(Argument):** $\theta = -\Phi = -\tan^{-1}(\frac{\omega}{k})$
    

### 3.2 복소수의 나눗셈과 편각의 관계

$w = \frac{1}{z}$ 꼴에서 $z = r e^{i\Phi}$라면, $w = \frac{1}{r} e^{-i\Phi}$가 됩니다. 즉, **분모의 위상($\Phi$)이 커질수록 전체 위상은 그만큼 뒤처지게(-)** 됩니다.

### 3.3 내적(Inner Product)을 활용한 데카르트 좌표계 풀이

극형식(Polar form) 변환 없이, 복소수 해를 직교 좌표계(Cartesian form)로 전개하여 푸는 대체 방법입니다.

1. **유리화:** $w = \frac{1}{1 + i(\omega/k)}$의 분모, 분자에 각각 $1 - i(\omega/k)$를 곱하여 실수부와 허수부를 분리합니다.
    
    $$w = \frac{1 - i(\omega/k)}{1 + (\omega/k)^2}$$
    
2. **실수부 추출:** $\hat{T}(t) = w \cdot e^{i\omega t}$를 전개한 후 실수부($\text{Re}$)만 추출하면 코사인과 사인의 합 형태로 나타납니다.
    
    $$T(t) = C \left[ 1 \cdot \cos(\omega t) + \left(\frac{\omega}{k}\right) \sin(\omega t) \right] \quad \left( 단, C = \frac{1}{1+(\omega/k)^2} \right)$$
    
3. **내적(Inner Product) 해석:** 대괄호 안의 식을 두 벡터 $\langle 1, \omega/k \rangle$와 $\langle \cos(\omega t), \sin(\omega t) \rangle$의 내적으로 해석합니다. 내적 공식에 의해 다음과 같이 삼각함수가 합성됩니다.
    
    $$1 \cdot \cos(\omega t) + \left(\frac{\omega}{k}\right) \sin(\omega t) = \sqrt{1^2 + \left(\frac{\omega}{k}\right)^2} \cos(\omega t - \Phi)$$
    
    _(이때 위상 지연 $\Phi = \tan^{-1}(\frac{\omega/k}{1}) = \tan^{-1}(\omega/k)$)_
    
4. **결론:** 위 식을 원래의 상수 $C$와 곱하여 정리하면, 극형식으로 접근했을 때와 완벽하게 동일한 진폭과 위상 지연을 가진 최종 해를 얻을 수 있습니다.

## 4. 최종 해 및 물리적 의미 (Physical Insights)

복소수 해 $\hat{T}(t)$를 다시 쓰면:

$$\hat{T}(t) = |w| e^{-i\Phi} e^{i\omega t} = \frac{1}{\sqrt{1 + (\frac{\omega}{k})^2}} e^{i(\omega t - \Phi)}$$

실수부만 취한 최종 해 $T(t)$는 다음과 같습니다.

$$T(t) = \frac{1}{\sqrt{1 + (\frac{\omega}{k})^2}} \cos(\omega t - \Phi)$$

### 4.1 주요 용어 정리

| **항목**                | **수식**                              | **의미**                             |
| --------------------- | ----------------------------------- | ---------------------------------- |
| **이득 (Gain)**         | $\frac{1}{\sqrt{1 + (\omega/k)^2}}$ | 입력 진폭 대비 출력 진폭의 비율 (**Amplitude**) |
| **위상 지연 (Phase Lag)** | $\Phi = \tan^{-1}(\omega/k)$        | 입력 피크와 출력 피크 사이의 각도 차이             |
| **시간 지연 (Time Lag)**  | $\Delta t = \frac{\Phi}{\omega}$    | 실제 시간상에서 출력이 늦어지는 정도               |

### 4.2 시스템 파라미터 $k$의 영향

$k$는 시스템의 **응답 속도**를 결정합니다.

- **$k \to \infty$ (매우 빠른 반응):**
    
    - 진폭(Gain) $\approx 1$: 입력을 거의 그대로 출력함.
        
    - $\Phi \approx 0$: 지연 없이 즉각적으로 반응함.
        
- **$k \to 0$ (매우 느린 반응):**
    
    - 진폭 $\approx 0$: 입력이 들어와도 시스템이 거의 움직이지 못함.
        
    - $\Phi \approx 90^\circ$: 최대 지연이 발생함.
