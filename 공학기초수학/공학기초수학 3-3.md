# 3-3. 자율 미분방정식 (Autonomous ODEs)

## 1. 정의 및 모델링 (Definition & Modeling)

### 1.1 자율 미분방정식이란?

독립변수 $t$가 식에 직접 나타나지 않고, 오직 종속변수 $y$에 의해서만 변화량이 결정되는 형태입니다.

$$\frac{dy}{dt} = f(y)$$

### 1.2 로지스틱 방정식 (Logistic Equation) 유도

현실적인 인구 성장이나 시스템 변화를 모델링할 때 사용합니다.

1. **성장률 설정:** $\frac{dy}{dt} = h(y)y$에서 $h(y)$를 성장률이라 함.
    
2. **가정:** * $y$가 작을 때는 자원이 풍부하여 성장률이 양수 ($h(y) > 0$).
    
    - $y$가 너무 커지면 자원 고갈로 성장률이 음수 ($h(y) < 0$).
        
3. **성장률 함수화:** 가장 간단한 선형 모델 $h(y) = A - By$를 적용.
    
4. **표준형 정리:**
    
    $$\frac{dy}{dt} = Ay - By^2 = Ay\left(1 - \frac{B}{A}y\right) = A\left(1 - \frac{y}{K}\right)y$$
    
    _(단, $K = A/B$이며 이를 환경 수용력, Carrying Capacity라 함)_
    

---

## 2. 평형 상태와 위상 분석 (Equilibrium & Phase Analysis)

방정식을 직접 풀지 않고 $\frac{dy}{dt}$와 $y$의 관계(이차함수 그래프)를 통해 해를 분석합니다.

### 2.1 평형점 (Equilibrium Points)

$\frac{dy}{dt} = f(y) = 0$이 되는 지점으로, 변화가 멈추는 상태입니다.

- **$y = 0$ (비안정 평형):** 인구가 없으면 변화가 없지만, 조금이라도 생기면 $0$에서 멀어짐.
    
- **$y = K$ (안정 평형):** 주변의 모든 해가 $K$라는 한계치로 수렴함.
    

### 2.2 구간별 해의 거동

- **$y < 0$:** $y' < 0$ (현실적으로는 고려하지 않음).
    
- **$0 < y < K$:** $y' > 0$ 이므로 $y$는 $K$를 향해 **증가**.
    
- **$y > K$:** $y' < 0$ 이므로 $y$는 $K$를 향해 **감소**.

## 3. 곡선의 모양과 오목/볼록 (Concavity Analysis)

해의 그래프($y(t)$)가 어떻게 휘어지는지 알기 위해 $2$계 도함수 $y''$를 분석합니다.

### 3.1 연쇄 법칙을 이용한 $y''$ 도출

$$y'' = \frac{d}{dt}\left(\frac{dy}{dt}\right) = \frac{d}{dt}f(y) = \frac{df}{dy} \cdot \frac{dy}{dt} = f'(y) \cdot f(y)$$

### 3.2 구간별 곡률 판단

- **$0 \le y \le K/2$ 구간:**
    
    - $f(y) > 0$ (증가 중), $f'(y) > 0$ (기울기 증가 중) $\implies y'' > 0$ (**아래로 볼록**)
        
- **$K/2 \le y \le K$ 구간:**
    
    - $f(y) > 0$ (증가 중), $f'(y) < 0$ (기울기 감소 중) $\implies y'' < 0$ (**위로 볼록**)
        
    - **결과:** $y = K/2$ 지점에서 변곡점(Inflection point)이 발생하여 $S$자 곡선을 형성함.
        
- **$y \ge K$ 구간:**
    
    - $f(y) < 0$ (감소 중), $f'(y) < 0$ (기울기 감소 중) $\implies y'' > 0$ (**아래로 볼록**)
        
    - _(참고: $y$가 $K$보다 클 때는 위에서 아래로 떨어지며 $K$에 수렴함)_
