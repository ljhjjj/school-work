
# 3-4. 2계 미분방정식의 도입 (Spring-Mass System)

이번 강의부터는 시스템의 상태 변화율(속도)뿐만 아니라, **가속도(2차 미분**)까지 포함된 2계 상미분방정식을 다룹니다. 교량의 진동, 행성의 궤도 운동, 자동차 서스펜션 등 수많은 물리 현상이 이 방정식으로 모델링됩니다.

## 1. 단순 스프링-질량 시스템 모델링 (Undamped System)

천장에 매달린 스프링에 질량 $m$이 달려있는 가장 단순한 시스템을 생각해 봅시다.

- **가정:** 공기 저항이나 마찰이 없는 이상적인 상태 (Undamped)
    
- **변수 정의:** 평형 상태(Equilibrium)에서의 위치를 $x=0$이라 하고, 아래쪽을 $+x$ 방향으로 잡습니다.
    
- **힘의 분석 (Newton's 2nd Law, $F=ma$):** 질량 $m$을 아래로 당겼다 놓으면, 스프링은 위로 원래 자리로 돌아가려는 복원력($-kx$)을 발생시킵니다.
    
    $$m \frac{d^2x}{dt^2} = -kx$$
    
- **표준형 2계 미분방정식 도출:** 양변을 $m$으로 나누고 이항하면 다음과 같은 운동 방정식(Equation of motion)이 나옵니다.
    
    $$\frac{d^2x}{dt^2} + \frac{k}{m}x = 0$$
    

---

## 2. 미분방정식의 해 (Solution to the ODE)

위 식을 만족하는 $x(t)$가 무엇인지 실험과 수학적 추론을 통해 찾아봅니다.

### 2.1 첫 번째 시도: $x = \cos(t)$

스프링이 위아래로 진동하는 모습을 보고 코사인 함수를 해로 가정해 봅니다.

- $x = \cos(t)$
    
- $x' = -\sin(t)$
    
- $x'' = -\cos(t)$
    

이를 식에 대입하면:

$-\cos(t) + \frac{k}{m}\cos(t) = 0 \implies \left(\frac{k}{m} - 1\right)\cos(t) = 0$ **결론:** 이 해는 $\frac{k}{m} = 1$이라는 아주 특수한 경우에만 성립하므로 일반적인 시스템의 해가 될 수 없습니다.

### 2.2 두 번째 시도: 주파수 인자($\omega_0$) 도입

스프링 상수 $k$와 질량 $m$에 따라 진동하는 속도가 다를 것이므로, 미지의 각주파수 $\omega_0$를 도입하여 보편적인 해 $x = \cos(\omega_0 t)$를 가정합니다.

- $x = \cos(\omega_0 t)$
    
- $x' = -\omega_0 \sin(\omega_0 t)$
    
- $x'' = -\omega_0^2 \cos(\omega_0 t)$
    

이를 식에 대입하면:

$-\omega_0^2 \cos(\omega_0 t) + \frac{k}{m}\cos(\omega_0 t) = 0$ $\left(\frac{k}{m} - \omega_0^2\right) \cos(\omega_0 t) = 0$

모든 시간 $t$에 대해 식이 성립하려면 괄호 안이 $0$이 되어야 합니다. **결론:** $\omega_0 = \sqrt{\frac{k}{m}}$ 일 때, $x = \cos(\omega_0 t)$는 미분방정식의 완벽한 해가 됩니다. 같은 논리로 $x = \sin(\omega_0 t)$ 역시 대입해 보면 해가 됨을 확인할 수 있습니다.

## 3. 왜 해가 두 개($\cos$와 $\sin$)의 합으로 나타나는가? (중요)


2계 미분방정식은 한 번 미분한 1계 방정식과 달리, 시스템을 완전히 결정짓기 위해 **두 가지 초기 조건(Initial Conditions**)이 필요합니다.

1. **초기 위치 (Initial Displacement, $x(0)$):** 처음에 스프링을 얼마나 잡아당겼는가?
    
2. **초기 속도 (Initial Velocity, $x'(0)$):** 처음에 어떤 속도로 밀었거나 찼는가?
    

교수님의 실험을 떠올려 봅시다.

- **Case 1: 처음에 잡아당겼다 가만히 놓았을 때 (초기 위치 O, 초기 속도 X)** 가장 높은 곳에서 출발해 내려오므로 **$\cos(\omega_0 t)$** 그래프를 그립니다.
    
- **Case 2: 평형 상태에서 위로 툭 쳤을 때 (초기 위치 X, 초기 속도 O)** $0$에서 출발해 위로 솟구치므로 **$\sin(\omega_0 t)$** 그래프를 그립니다.
    

**[일반적인 상황]** 현실에서는 스프링을 "어느 정도 잡아당긴 상태에서(초기 위치) 밀어버리는(초기 속도)" 일이 발생합니다. 즉, 두 가지 상황이 **중첩(Superposition**)되어 나타납니다. 따라서 2계 미분방정식의 가장 보편적인 '일반해(General Solution)'는 이 두 가지 독립적인 움직임(해)을 상수배하여 더한 형태가 되어야만 모든 물리적 상황을 설명할 수 있습니다.

$$x(t) = A \cos(\omega_0 t) + B \sin(\omega_0 t)$$

이 식은 3-2에서 배웠던 삼각함수의 합성을 통해 하나의 진폭과 위상을 가진 함수로 표현할 수 있습니다.

$$x(t) = C \cos(\omega_0 t - \Phi) \quad (\text{단, } C = \sqrt{A^2 + B^2}, \, \Phi = \tan^{-1}(B/A))$$

## 4. 댐핑(Damping)이 추가된 현실적 모델

이제 진공 상태가 아니라, 물이나 꿀 속(혹은 공기 저항이 있는 곳)에서 진동하는 현실적인 스프링 시스템을 모델링합니다.

- **저항력 (Damping Force):** 속도($x'$)에 비례하여 운동을 방해하는 힘입니다. 즉, $-cx'$로 표현됩니다. ($c$는 댐핑 상수)
    
- **힘의 분석:** $m x'' = -kx - cx'$
    
- **표준형 도출:**
    
    $$m x'' + cx' + kx = 0$$
    
    양변을 $m$으로 나누어 정리하면 우리가 앞으로 다루게 될 가장 전형적인 2계 상미분방정식의 형태가 나옵니다.
    
    $$x'' + \frac{c}{m}x' + \frac{k}{m}x = 0$$
    
    (수학적으로는 변수를 간단히 하여 $y'' + ay' + by = 0$ 형태로 표기합니다.)
