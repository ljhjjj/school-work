
# 3-5. 2계 선형 ODE: 상수 계수를 갖는 제차 방정식

## 1. 표준형과 해의 가정

우리는 다음과 같은 상수 계수($a, b$)를 갖는 2계 선형 제차 미분방정식을 해결합니다.

$$y'' + ay' + by = 0 \quad \cdots (*)$$

### 1.1 왜 $y = c_1y_1 + c_2y_2$ 형태인가? (물리적/수학적 이유)

질문하신 "물리적 현상으로 인해 두 가지 식을 가진다"는 의미는 다음과 같습니다.

1. **자유도의 수:** 2계 미분방정식은 가속도($y''$)를 다룹니다. 물체의 미래 위치를 알기 위해서는 **초기 위치($y(0)$**)와 **초기 속도($y'(0)$**)라는 두 가지 독립적인 정보가 필요합니다.
    
2. **수학적 독립성:** 이 두 가지 초기 조건을 모두 만족시키려면, 서로 겹치지 않는(독립적인) 두 개의 기초 해($y_1, y_2$)가 존재해야 하며, 이들의 조합으로 일반해를 구성해야 합니다.
    

---

## 2. 특성방정식 (Characteristic Equation)

해의 형태를 $y = e^{\lambda t}$라고 가정하고 $(*)$ 식에 대입하면 다음의 **특성방정식**을 얻습니다.

$$\lambda^2 + a\lambda + b = 0$$

이 방정식은 이차방정식이므로 판별식 $D = a^2 - 4b$의 부호에 따라 세 가지 케이스로 나뉩니다.

---

## 3. Case 1: 서로 다른 두 실근 ($D > 0$)

특성방정식이 $\lambda_1, \lambda_2$라는 두 개의 서로 다른 실근을 가질 때입니다.

**예제: $y'' + 3y' + 2y = 0$, 초기 조건 $y(0)=1, y'(0)=2$**

1. **특성방정식:** $\lambda^2 + 3\lambda + 2 = 0 \rightarrow (\lambda+1)(\lambda+2)=0$
    
2. **기초 해:** $y_1 = e^{-t}, y_2 = e^{-2t}$
    
3. **일반해:** $y = c_1 e^{-t} + c_2 e^{-2t}$
    
4. **상수 결정 (IC 적용):**
    
    - $y(0) = c_1 + c_2 = 1$
        
    - $y'(0) = -c_1 - 2c_2 = 2$
        
    - 연립하여 풀면 $c_1 = 4, c_2 = -3$
        
    - **최종해:** $y(t) = 4e^{-t} - 3e^{-2t}$
        

---

## 4. Case 2: 중근 ($D = 0$) - 계수 감소법

특성방정식이 $\lambda = -a/2$라는 하나의 근만 가질 때입니다. 이때 $y_1 = e^{\lambda t}$는 알지만, 두 번째 독립적인 해 $y_2$가 부족합니다. 이를 찾기 위해 **계수 감소법(Reduction of Order**)을 사용합니다.

### 4.1 $y_2$ 도출 과정

1. **가정:** $y_2 = u(t) y_1(t) = u(t) e^{-\frac{a}{2}t}$
    
2. **미분 및 대입:** $y_2$의 1차, 2차 미분을 원래 식 $(*)$에 대입하여 정리하면 다음과 같은 식을 얻습니다.
    
    $$e^{-\frac{a}{2}t} \left[ u'' + \left(b - \frac{a^2}{4}\right)u \right] = 0$$
    
3. **조건 활용:** 중근 조건에서 $D = a^2 - 4b = 0$이므로, 괄호 안의 $b - \frac{a^2}{4}$는 $0$이 됩니다.
    
4. **결과:** $u'' = 0 \rightarrow u(t) = k_1 t + k_2$
    
5. **최종 선택:** 가장 단순한 형태를 위해 $k_1=1, k_2=0$을 택하면 $u=t$가 됩니다.
    

**중근일 때의 일반해:**

$$y = c_1 e^{-\frac{a}{2}t} + c_2 t e^{-\frac{a}{2}t}$$

**예제: $y'' + 4y' + 4y = 0$**

- 특성방정식: $(\lambda+2)^2 = 0 \rightarrow \lambda = -2$ (중근)
    
- 일반해: $y = c_1 e^{-2t} + c_2 t e^{-2t}$
