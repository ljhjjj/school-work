# 4-2. 오일러 방정식 (Euler's Equation)

이번 강의에서는 상수 계수 미분방정식과 형태는 다르지만, 특정한 치환을 통해 쉽게 풀 수 있는 **오일러 방정식(Euler's equation)**에 대해 알아봅니다.

## 1. 오일러 방정식의 기본 형태

오일러 방정식은 독립 변수 $x$의 거듭제곱 차수와 도함수의 미분 차수가 일치하는 아래와 같은 형태를 가집니다.

$$x^2 y'' + pxy' + qy = 0$$

(단, $p$와 $q$는 상수입니다. 필기본의 $\lambda^2 y''$는 $x^2 y''$의 오기재입니다.)

## 2. 해의 가정과 대입

상수 계수 방정식에서 $y = e^{\lambda t}$를 가정했던 것과 달리, 오일러 방정식에서는 변수가 $x$의 거듭제곱 형태이므로 해를 **$y = x^\alpha$** 로 가정합니다.

이를 미분하면 다음과 같습니다.

- $y = x^\alpha$
    
- $y' = \alpha x^{\alpha-1}$
    
- $y'' = \alpha(\alpha-1)x^{\alpha-2}$
    

이 도함수들을 원래의 오일러 방정식에 대입합니다.

$$x^2 (\alpha(\alpha-1)x^{\alpha-2}) + px (\alpha x^{\alpha-1}) + q(x^\alpha) = 0$$

모든 항이 공통으로 $x^\alpha$를 가지게 되므로, 식을 묶어서 정리할 수 있습니다.

$$\left[\alpha(\alpha-1) + p\alpha + q\right] x^\alpha = 0$$

$$(\alpha^2 - \alpha + p\alpha + q) x^\alpha = 0$$

## 3. 특성방정식 도출

미분방정식이 성립하기 위해 $x \neq 0$이라고 가정하면, 괄호 안의 식이 $0$이 되어야 합니다. 이를 $\alpha$에 대해 정리하면 오일러 방정식의 **특성방정식**을 얻을 수 있습니다.

$$\alpha^2 + (p-1)\alpha + q = 0$$

이 2차 방정식을 근의 공식을 통해 풀면 $\alpha$ 값을 구할 수 있습니다.

$$\alpha = \frac{-(p-1) \pm \sqrt{(p-1)^2 - 4q}}{2}$$

_(필기본에는 분자가 $-(p+1)$로 적혀있으나, 식 전개 상 $-(p-1)$이 정확한 수식입니다.)_

## 4. 일반해 (서로 다른 두 실근의 경우)

교수님께서는 이번 학기 과정에서는 오일러 방정식의 해가 중근이나 복소수 근을 가지는 경우는 제외하고, **서로 다른 두 실근($\alpha_1, \alpha_2$)을 가지는 가장 단순한 경우만 다룬다**고 명시하셨습니다.

서로 다른 두 실근 $\alpha_1, \alpha_2$가 도출되었다면, 일반해는 두 해의 선형 결합으로 나타납니다.

$$y = C_1 x^{\alpha_1} + C_2 x^{\alpha_2}$$
