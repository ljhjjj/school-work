# 5-2. 특수해 구하기 3번째 방법: 복소수화 기법 (Complexification)

이 방법은 우변의 함수가 지수함수와 삼각함수($\sin, \cos$)의 곱으로 이루어져 있을 때, 미정계수법보다 훨씬 간단하고 우아하게 특수해를 구하는 수학적 기교입니다.

## 1. 복소수화(Complexification)의 기본 아이디어

우리가 풀고자 하는 일반적인 비제차 방정식이 다음과 같다고 가정해 봅시다.

$$y'' + ay' + by = Ce^{\alpha t} \cos(\omega t) \quad \text{또는} \quad Ce^{\alpha t} \sin(\omega t)$$

이를 오일러 공식을 활용하여 하나의 **복소수 지수함수**를 우변으로 가지는 새로운 미분방정식으로 변환(Complexify)합니다.

$$\hat{y}'' + a\hat{y}' + b\hat{y} = Ce^{st}$$

(여기서 $s = \alpha + i\omega$ 인 복소수입니다.)

우변이 지수함수 형태이므로, 이 복소수 미분방정식의 특수해 역시 다음과 같은 형태일 것이라고 가정합니다.

$$\hat{y}_p = \tilde{y}e^{st}$$

(여기서 $\tilde{y}$는 미지의 복소수 상수입니다.)

## 2. 미분 연산자(Differential Operator)와 치환 공식

계산을 단순화하기 위해 미분 연산자 $D = \frac{d}{dt}$ 를 도입합니다. ($D^2 = \frac{d^2}{dt^2}$가 됩니다.)
    
- 다항 연산자: $P(D) = D^2 + aD + b$
    

이를 이용하면 복소수화된 미분방정식을 아주 간단하게 쓸 수 있습니다.

$$P(D)\hat{y} = Ce^{st}$$

**[핵심 치환 공식 (Substitution Formula)]** 지수함수 $e^{st}$에 미분 연산자 $P(D)$를 적용하면, 미분할 때마다 $s$가 앞으로 튀어나오므로 단순히 $D$ 자리에 $s$를 대입한 것과 같은 결과가 나옵니다.

$$P(D)e^{st} = P(s)e^{st}$$

## 3. 특수해 도출 공식 (Exponential Response Formula)

가정한 해 $\hat{y}_p = \tilde{y}e^{st}$ 를 방정식에 대입하고 치환 공식을 적용합니다.

$$P(D)(\tilde{y}e^{st}) = \tilde{y}P(D)e^{st} = \tilde{y}P(s)e^{st} = Ce^{st}$$

양변에서 $e^{st}$를 약분하여 미지의 상수 $\tilde{y}$를 구합니다.

$$\tilde{y} = \frac{C}{P(s)}$$

### 일반적인 경우: $P(s) \neq 0$

특수해 $\hat{y}_p$는 다음과 같이 단번에 도출됩니다.

$$\mathbf{\hat{y}_p = \frac{C}{P(s)} e^{st}}$$

- **최종 해 구하기:** * 원래 문제의 우변이 코사인($\cos$)이었다면, 구한 $\hat{y}_p$의 **실수부(Real part)** 를 취합니다.
    
    - 원래 문제의 우변이 사인($\sin$)이었다면, **허수부(Imaginary part)** 를 취하면 최종 특수해 $Y_p$가 됩니다.
        

실제 풀이 예시는 [부록 5-2-1: 복소수화 기법 풀이 예시](부록/5-2-1%20복소수화%20기법%20풀이%20예시.md)를 참고하시길 바랍니다.

## 4. 예외 케이스: $P(s) = 0$ 인 경우 (공진 현상)

만약 $s$가 특성방정식 $P(s) = 0$의 근(Root)과 일치한다면 분모가 0이 되어 위의 공식을 사용할 수 없습니다. 이러한 경우 미정계수법의 Modification rule에서 $t$를 곱했던 것과 동일한 원리로 다음 공식을 사용합니다.

- **Simple root (단일근)인 경우:** 해에 $t$를 곱하고, 분모는 $P(s)$를 한 번 미분한 $P'(s)$를 사용합니다.
    
    $$\mathbf{\hat{y}_p = \frac{C t e^{st}}{P'(s)}}$$
    
- **Double root (중근)인 경우:** 해에 $t^2$을 곱하고, 분모는 두 번 미분한 $P''(s)$를 사용합니다.
    
    $$\mathbf{\hat{y}_p = \frac{C t^2 e^{st}}{P''(s)}}$$
    
이 예외 케이스가 성립하는 이유는 [공학기초수학 6-1](공학기초수학%206-1.md)에서 다뤄집니다.
