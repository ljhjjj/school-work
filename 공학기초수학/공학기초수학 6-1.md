# 6-1. 지수 이동 정리(Exponential Shift Theorem)와 단일근 공식의 상세 증명

이 노트는 우변이 $f(t) = t \cos t$ 와 같이 다항함수와 삼각함수의 곱으로 이루어져 있거나, 특성방정식의 근과 우변의 복소수 $s$가 겹치는 공진 상태($P(s)=0$)일 때 특수해 공식을 수학적으로 엄밀하게 증명하는 과정을 담고 있습니다.

## 1. 특수해의 형태 가정

우변의 함수가 $f(t) = t \cos t$ 일 때, 복소수화 기법을 적용하면 우변은 $t e^{st}$ 형태가 됩니다. 이때 특수해는 다항식과 지수함수의 곱인 다음과 같은 형태로 가정할 수 있습니다.

$$\hat{y}_p = (K_1 + tK_2)e^{st}$$

이를 다항 연산자 $P(D)$ 형태의 미분방정식에 대입하면 우리가 풀어야 할 기본 식은 다음과 같이 세워집니다.

$$P(D)(K_1 t + K_2)e^{st} = t e^{st}$$

## 2. 지수 이동 정리 (Exponential Shift Theorem) 상세 증명

위와 같이 지수함수 $e^{st}$와 임의의 함수 $u(t)$가 곱해진 형태를 미분할 때, 지수함수를 연산자 앞으로 빼내어 계산을 단순화하는 정리가 바로 '지수 이동 정리'입니다.

$$P(D)[e^{st}u(t)] = e^{st}P(D+s)u(t)$$

이 정리가 성립하는지 1계 미분($D$)과 2계 미분($D^2$)에 대해 각각 상세히 증명해 봅니다.

### ① 1계 미분 연산자 ($P(D) = D$) 증명

곱의 미분법을 적용하여 전개한 뒤 $e^{st}$로 묶어냅니다.

$$D(e^{st}u(t)) = s e^{st}u(t) + e^{st}Du(t) = e^{st}(D+s)u(t)$$

### ② 2계 미분 연산자 ($P(D) = D^2$) 증명

$D^2$은 $D$를 두 번 적용한 것이므로, 방금 구한 1계 미분 결과를 활용합니다.

$$D^2(e^{st}u(t)) = D(D(e^{st}u(t))) = D(e^{st}(D+s)u(t))$$

계산의 편의를 위해 일시적으로 $(D+s)u(t) = v(t)$ 로 치환합니다.

$$= D(e^{st}v(t))$$

여기에 다시 1계 미분 결과를 적용하면 다음과 같이 지수함수가 앞으로 빠져나옵니다.

$$= e^{st}(D+s)v(t)$$

마지막으로 $v(t)$ 자리에 원래 식인 $(D+s)u(t)$ 를 다시 대입하여 전개합니다.

$$= e^{st}(D+s)(D+s)u(t) = e^{st}(D+s)^2 u(t)$$

이로써 연산자의 차수에 상관없이 지수 이동 정리가 성립함이 완벽히 증명되었습니다.

## 3. 공진 상태 ($P(s)=0$)의 단일근 특수해 공식 상세 증명

앞서 5-3 노트에서 우변의 $s$가 특성방정식의 단일근일 때, 특수해가 $\hat{y}_p = \frac{te^{st}}{P'(s)}$ 가 된다고 배웠습니다. 이 공식을 방금 유도한 지수 이동 정리를 통해 엄밀하게 증명합니다.

우리가 증명해야 할 목표 수식은 다음과 같습니다.

$$P(D)\left[ \frac{te^{st}}{P'(s)} \right] = e^{st}$$

### ① 좌변에 지수 이동 정리 적용 (분자 전개)

좌변의 분자에 있는 $e^{st} \cdot t$ 부분에 지수 이동 정리를 적용하여 $e^{st}$를 연산자 앞으로 빼냅니다 (이때 $u(t)=t$ 입니다).

$$\frac{e^{st}P(D+s)t}{P'(s)} = e^{st}$$

이제 주어진 연산자 $P(D)$가 단일근 $s$와 또 다른 근 $\delta$를 가진다고 가정하면, $P(D) = (D-s)(D-\delta)$ 로 쓸 수 있습니다. 분자의 연산자 $P(D+s)$에 이를 대입하여 $t$에 적용해 봅니다.

$$P(D+s)t = ((D+s)-s)((D+s)-\delta)t = D(D+s-\delta)t$$

이를 전개하여 미분을 수행합니다. ($Dt = 1$ 이고, 상수를 미분하면 $0$입니다.)

$$= (D^2 + (s-\delta)D)t = D^2(t) + (s-\delta)D(t) = 0 + (s-\delta) \cdot 1 = s-\delta$$

따라서 좌변 분자의 연산 결과는 **$s-\delta$** 가 됩니다.

### ② 분모의 $P'(s)$ 전개

이제 분모에 있는 $P'(s)$가 무엇인지 계산해 봅니다. $P(D) = (D-s)(D-\delta)$ 를 곱의 미분법을 이용해 미분합니다.

$$P'(D) = (D-\delta) + (D-s)$$

이 식의 $D$ 자리에 $s$를 대입하면 뒤의 항은 $s-s=0$이 되어 사라집니다.

$$P'(s) = (s-\delta) + (s-s) = s-\delta$$

따라서 분모의 값 역시 **$s-\delta$** 가 됩니다.

### ③ 최종 증명 완료

①에서 구한 분자 값과 ②에서 구한 분모 값을 원래 식에 대입합니다.

$$\frac{e^{st}(s-\delta)}{(s-\delta)} = e^{st}$$

분자와 분모의 $(s-\delta)$ 가 완벽하게 약분되어 사라지고, 우변인 $e^{st}$ 와 완전히 동일해짐으로써  
$$P(D)\left[ \frac{te^{st}}{P'(s)} \right] = e^{st}$$
단일근 특수해 공식이 참임이 증명되었습니다.
