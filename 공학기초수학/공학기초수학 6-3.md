# 6-3. 중첩의 원리와 합성곱 적분

## 1. 중첩의 원리 (Principle of Superposition)

- 복잡한 형태의 외력(Forcing function)이 주어졌을 때, 이를 여러 개의 단순한 외력들의 합으로 쪼개어 생각할 수 있습니다.
    
- 선형 미분방정식 연산자 $L$에 대하여 다음의 합의 법칙(Sum rule)이 성립합니다.
    
    - $L(y_1) = f_1$
        
    - $L(y_2) = f_2$
        
    - $\implies L(y_1 + y_2) = f_1 + f_2$
        
- 따라서 전체 외력이 $F = f_1 + f_2 + f_3 + f_4$ 와 같이 주어진다면, 전체 시스템의 응답은 각각의 단일 외력에 대한 응답을 모두 합친 $Y = y_1 + y_2 + y_3 + y_4$ 가 됩니다.
    

## 2. 임펄스(Impulse)와 단위 임펄스 응답(Unit Impulse Response)

- **임펄스 (Impulse, $\delta$):** 특정 시간에 아주 짧고 강하게 가해지는 충격(Kick)을 의미합니다. 시간 $t_1$에서 가해진 임펄스는 디랙 델타 함수 기호를 사용하여 $\delta(t - t_1)$으로 표기합니다.
    
- **임펄스 응답 (Impulse Response, $w$):** 시스템에 단위 임펄스가 가해졌을 때 나타나는 시스템의 고유한 반응입니다.
    
    - 시간 $t_0$에 임펄스가 가해졌다면, 그에 따른 시스템의 응답은 $w(t - t_0)$의 형태로 나타납니다.
        
- 아무리 복잡한 입력 신호라도 무수히 많은 임펄스들의 합으로 간주할 수 있으며, 전체 응답 역시 각각의 임펄스가 만들어낸 응답($y_1, y_2 \dots$)들을 겹겹이 더하여(Superposition) 구할 수 있습니다.
    

## 3. 연속적인 외력에 대한 근사 (Approximation of Continuous Force)

- 곡선 형태의 연속적이고 복잡한 외력 $f(t)$를 아주 짧은 시간 간격 $\Delta t$를 밑변으로 하는 여러 개의 직사각형(상수 함수) 조각들의 합으로 근사할 수 있습니다.


- 특정 시간 $t_i$에 해당하는 직사각형 면적(즉, 충격의 크기)은 $f(t_i) \Delta t$ 가 됩니다.
    
- 이 얇은 직사각형 조각 하나가 시스템에 유발하는 응답은, 크기가 $f(t_i)\Delta t$ 이고 $t_i$ 시점에서 발생한 임펄스 응답과 동일하므로 다음과 같이 수식화할 수 있습니다.
    
    $$y_i(t) \approx f(t_i) w(t - t_i) \Delta t$$
    
- 결과적으로 임의의 시간 $t$에서의 전체 시스템 응답 $Y(t)$는, 측정 시점 이전까지 차례대로 가해진 모든 조각들의 응답을 합산한 값이 됩니다.
    
    $$Y(t) \approx \sum_{i=1}^{n-1} Y(t_i) = \sum_{i=1}^{n-1} f(t_i) w(t - t_i) \Delta t$$

## 4. 합성곱 적분 (Convolution Integral)

- 위에서 구한 시그마($\sum$) 근사식에서 시간 간격 $\Delta t$를 무한히 작게 쪼개어 한계(Limit)로 보내면($\Delta t \to 0$), 이 합산식은 정확한 정적분 형태인 **합성곱 적분(Convolution Integration)**으로 변환됩니다.
    
    $$\lim_{\Delta t \to 0} \sum \implies \int_{0}^{t} f(\tau) w(t - \tau) d\tau = Y(t)$$
    
- **물리적 의미:** 현재 시간 $t$에서의 전체 상태 $Y(t)$는, 과거의 특정 시점(시간 $\tau$)들에 작용했던 모든 외력 $f(\tau)$들이 시간이 흐름에 따라 감쇠되며 현재까지 미치고 있는 잔여 영향력($w(t-\tau)$)을 시작점부터 현재 시점까지 연속적으로 누적(적분)한 총합을 의미합니다.
