# 6-4. 라플라스 변환 (Laplace Transform)

## 1. 라플라스 변환의 정의 (Definition)

시간 $t$에 대한 함수 $f(t)$의 라플라스 변환은 $t$ 영역의 함수를 복소수 $s$ 영역의 함수 $F(s)$로 변환하는 적분 연산입니다.

- **정의식:**
    
    $$L\{f(t)\} = F(s) = \int_{0}^{\infty} f(t)e^{-st} dt$$
    
- 적분 구간이 $0$부터 $\infty$까지이므로, 이는 무한대에서의 극한을 포함하는 **이상적분(Improper Integration)**입니다.
    
- **수렴 조건 (Existence):** 이 적분이 발산하지 않고 특정 값으로 수렴해야만 변환이 존재합니다. 이를 위해 $f(t)$는 $t \to \infty$ 일 때 지수함수 $e^{-st}$에 의해 $0$으로 억눌릴 수 있는 **지수 형태(Exponential type)**여야 합니다 ($|f(t)| \le Me^{kt}$).
    

## 2. 주요 성질 (Properties)

- **선형성 (Linearity):** 상수배와 덧셈에 대해 분배가 성립합니다.
    
    $$L\{c f(t)\} = c L\{f(t)\}$$
    
    $$L\{f_1(t) + f_2(t)\} = L\{f_1(t)\} + L\{f_2(t)\}$$
    
- **$s$-이동 정리 ($s$-Shifting):** 함수 $f(t)$에 지수함수 $e^{at}$가 곱해지면, $s$ 영역에서는 $s$축으로 $a$만큼 평행 이동하는 결과가 나타납니다.
    
    $$L\{e^{at}f(t)\} = F(s-a)$$
    

## 3. 기본 함수의 라플라스 변환 표 (Table of Transforms)

강의에서 유도한 기본 함수들의 라플라스 변환은 다음과 같습니다. (이는 미분방정식을 풀 때 변환 및 역변환 테이블로 사용됩니다.)

| **f(t)**         | **F(s)=L{f(t)}**                                       | **비고 및 유도 방법**                                                                                |
| ---------------- | ------------------------------------------------------ | --------------------------------------------------------------------------------------------- |
| $1$              | $\frac{1}{s}$                                          | 정의에 대입하여 $\lim_{R \to \infty} \int_0^R e^{-st}dt$ 계산                                          |
| $e^{at}$         | $\frac{1}{s-a}$                                        | $s$-shifting 정리 적용 ($f(t)=1$에 $e^{at}$ 곱함)                                                    |
| $t^n$            | $\frac{n!}{s^{n+1}}$                                   | 부분적분법(Integration by parts)을 $n$번 반복 적용                                                       |
| $e^{i\omega t}$  | $\frac{1}{s-i\omega} = \frac{s+i\omega}{s^2+\omega^2}$ | $e^{at}$의 변환에서 $a = i\omega$ 대입 후 유리화                                                         |
| $\cos(\omega t)$ | $\frac{s}{s^2+\omega^2}$                               | $e^{i\omega t}$ 변환 결과의 **실수부(Real part)** 추출 ($e^{i\omega t} = \cos\omega t + i\sin\omega t$) |
| $\sin(\omega t)$ | $\frac{\omega}{s^2+\omega^2}$                          | $e^{i\omega t}$ 변환 결과의 **허수부(Imaginary part)** 추출                                             |

## 4. 도함수의 라플라스 변환 (Transforms of Derivatives)

미분방정식을 대수방정식으로 바꾸기 위한 핵심 공식입니다. $t$ 영역의 '미분' 연산이 $s$ 영역에서는 '$s$를 곱하는 연산'으로 바뀝니다. (부분적분을 통해 유도.)

- **1계 도함수:**
    
    $$L\{y'\} = sY(s) - y(0)$$
    
- **2계 도함수:**
    
    $$L\{y''\} = s^2Y(s) - s y(0) - y'(0)$$
    

## 5. 미분방정식 풀이 절차 및 장점 (Solving ODEs)

라플라스 변환을 이용해 $y'' + ay' + by = f(t)$ 와 같은 미분방정식을 푸는 과정은 다음과 같습니다.

1. **라플라스 변환 (Transform):** 양변에 라플라스 변환을 취해 $t$ 공간의 미분방정식을 $s$ 공간의 단순한 대수 방정식(Algebraic Equation)으로 바꿉니다.
    
2. **대수적 풀이 (Algebra):** 식을 $Y(s)$ 에 대해 정리합니다. ($Y(s) = \frac{P_1(s)}{P_2(s)}$ 형태의 다항식 분수 꼴이 됩니다.)
    
3. **부분분수 전개 (Partial Fractions):** 구한 $Y(s)$를 변환표에 있는 기본 형태들의 합으로 쪼갭니다.
    
4. **역 라플라스 변환 (Inverse Transform):** 변환표를 역으로 적용하여 $Y(s)$를 다시 $t$ 공간의 해 $y(t)$로 되돌립니다.
    

풀이법 예제는 다음 부록[6-4-1 라플라스변환 IVP 풀이 예시](부록/6-4-1%20라플라스변환%20IVP%20풀이%20예시.md)를 참고하시길 바랍니다.

**✅ 라플라스 변환 풀이법의 장점**

- **제차해($y_h$)와 특수해($y_p$)를 한 번에 동시에 구합니다.** (따로 구할 필요 없음)
    
- **초기 조건($y(0), y'(0)$)이 도함수 변환 과정에서 자동으로 식에 반영됩니다.** (나중에 미분해서 계수를 찾을 필요 없음)
    
- **간헐적이거나 복잡한 외부 입력($f(t)$)도 쉽게 다룰 수 있습니다.** (합성곱(Convolution)과 연계 시 매우 강력함)
    

## 6. 합성곱 정리 (Convolution Theorem)

$s$ 공간에서의 곱셈 연산은 $t$ 공간에서의 합성곱(Convolution) 연산과 완벽하게 대응됩니다.

- **공식:**
    
    $$L\{f * g\} = F(s)G(s)$$
    
    _(여기서 $f*g = \int_0^t f(u)g(t-u)du$ 입니다.)*
    
- **물리적 응용:** $Y(s) = F(s)W(s)$ (외력 $\times$ 시스템 응답) 형태를 역변환하면, 앞서 배웠던 합성곱 적분인 $y(t) = \int_0^t f(\tau)w(t-\tau)d\tau$ 가 수학적으로 정확히 유도됩니다.
