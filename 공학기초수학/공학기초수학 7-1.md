# 공학기초수학 7-1 라플라스 변환 추가 정리

## 1. $f(t) = \frac{1}{\sqrt{t}}$의 라플라스 변환 증명

라플라스 변환의 정의에 따라 식을 세우고, 적분 변수를 치환하여 가우스 적분(Gaussian Integral) 형태로 변환하여 푸는 과정입니다.

$$L\left(\frac{1}{\sqrt{t}}\right) = \int_{0}^{\infty} t^{-\frac{1}{2}} e^{-st} dt$$

**1) 적분 변수 치환:**

$t = x^2$ 로 치환하면, $dt = 2x dx$ 가 됩니다.

$$= \int_{0}^{\infty} (x^2)^{-\frac{1}{2}} e^{-s(x^2)} (2x dx) = 2 \int_{0}^{\infty} e^{-sx^2} dx$$

**2) 가우스 적분 $A$의 계산:**

$A = \int_{0}^{\infty} e^{-x^2} dx$ 라고 가정하고, $A^2$을 구하기 위해 극좌표계로 치환합니다.

$$A^2 = \int_{0}^{\infty} e^{-x^2} dx \int_{0}^{\infty} e^{-y^2} dy = \int_{0}^{\infty} \int_{0}^{\infty} e^{-(x^2+y^2)} dx dy$$

- **극좌표 치환:** $x = r\cos\theta, y = r\sin\theta$
    
- **적분 범위 (1사분면):** $x, y \ge 0$ 이므로 $r \in [0, \infty)$, $\theta \in [0, \frac{\pi}{2}]$
    
- **야코비안(Jacobian) 행렬식:**
    
    $$\left| \frac{\partial(x,y)}{\partial(r,\theta)} \right| = \begin{vmatrix} \frac{\partial x}{\partial r} & \frac{\partial x}{\partial \theta} \\ \frac{\partial y}{\partial r} & \frac{\partial y}{\partial \theta} \end{vmatrix} = \begin{vmatrix} \cos\theta & -r\sin\theta \\ \sin\theta & r\cos\theta \end{vmatrix} = r\cos^2\theta + r\sin^2\theta = r$$
    

**전체 평면에 대한 적분 $B$를 먼저 계산합니다:**

$$B = \int_{r=0}^{\infty} \int_{\theta=0}^{2\pi} r e^{-r^2} d\theta dr = 2\pi \int_{0}^{\infty} r e^{-r^2} dr = 2\pi \left[ -\frac{1}{2} e^{-r^2} \right]_{0}^{\infty} = \pi$$

**$A$값 도출:**

$A^2$는 1사분면에 해당하므로 $B$의 $\frac{1}{4}$입니다.

$$A^2 = \frac{B}{4} = \frac{\pi}{4} \implies A = \frac{\sqrt{\pi}}{2}$$

**3) 최종 변환 결과 도출:**

원래의 라플라스 변환 식으로 돌아와 정리합니다.

$$2 \int_{0}^{\infty} e^{-sx^2} dx$$

여기서 $\sqrt{s}x = u$ 로 치환하면 $dx = \frac{du}{\sqrt{s}}$ 가 됩니다.

$$= 2 \int_{0}^{\infty} e^{-u^2} \frac{du}{\sqrt{s}} = \frac{2}{\sqrt{s}} \int_{0}^{\infty} e^{-u^2} du = \frac{2}{\sqrt{s}} A = \frac{2}{\sqrt{s}} \frac{\sqrt{\pi}}{2} = \sqrt{\frac{\pi}{s}}$$

**결론:**

$$L\left(\frac{1}{\sqrt{t}}\right) = \sqrt{\frac{\pi}{s}}$$

## 2. 합성곱(Convolution)의 라플라스 변환 증명

두 함수 $f$와 $g$의 합성곱 적분에 라플라스 변환을 취하면, 각각의 라플라스 변환의 곱인 $F(s)G(s)$가 됨을 증명하는 과정입니다.

$$L\left( \int_{0}^{t} f(\tau)g(t-\tau)d\tau \right) = \int_{t=0}^{\infty} \int_{\tau=0}^{t} f(\tau)g(t-\tau)d\tau \, e^{-st} dt$$

**1) 적분 순서 변경 (Change of Integration Order):**

적분 영역을 $t\tau$-평면에서 보면 $\tau \le t \le \infty$, $0 \le \tau \le \infty$ 이므로 순서를 바꿉니다.

$$= \int_{\tau=0}^{\infty} f(\tau) \left( \int_{t=\tau}^{\infty} g(t-\tau) e^{-st} dt \right) d\tau$$

**2) 내부 적분 변수 치환:**

내부 적분에서 $\xi = t-\tau$ 로 치환합니다.

그러면 $t = \xi+\tau$ 가 되고, $dt = d\xi$ 가 됩니다.

- 적분 범위 변화: $t$가 $\tau \to \infty$ 로 갈 때, $\xi$는 $0 \to \infty$ 로 갑니다.
    

$$= \int_{\tau=0}^{\infty} f(\tau) \left( \int_{\xi=0}^{\infty} g(\xi) e^{-s(\xi+\tau)} d\xi \right) d\tau$$

**3) 지수법칙 적용 및 적분 분리:**

$e^{-s(\xi+\tau)} = e^{-s\xi} e^{-s\tau}$ 이므로, 각각의 변수에 맞게 식을 분리합니다.

$$= \int_{\tau=0}^{\infty} f(\tau) e^{-s\tau} d\tau \int_{\xi=0}^{\infty} g(\xi) e^{-s\xi} d\xi$$

**결론:**

각각의 적분이 $L{f(t)}$와 $L{g(t)}$의 정의식과 완벽하게 일치하므로 다음이 성립합니다.

$$L(f * g) = F(s)G(s)$$
