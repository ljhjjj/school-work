
# 예제: 라플라스 변환을 이용한 초기값 문제(IVP) 풀이

### 1. 문제 정의

다음과 같은 비제차 선형 미분방정식과 초기 조건이 주어졌습니다.

- **미분방정식:** $y'' - y = e^{-t}$
    
- **초기 조건:** $y(0) = 1, y'(0) = 0$
    

### 2. 풀이 단계

#### 1단계: 방정식의 양변에 라플라스 변환 취하기

라플라스 변환의 선형성을 이용하여 각 항을 변환합니다.

$$L\{y''\} - L\{y\} = L\{e^{-t}\}$$

#### 2단계: 도함수의 변환 공식 및 초기 조건 대입

도함수의 변환 공식($L\{y''\} = s^2 Y(s) - s y(0) - y'(0)$)과 우변의 변환($L\{e^{-at}\} = \frac{1}{s+a}$)을 적용합니다.

$$(s^2 Y(s) - s \cdot 1 - 0) - Y(s) = \frac{1}{s+1}$$

#### 3단계: $Y(s)$에 대한 대수 방정식으로 정리

식을 정리하여 $Y(s)$를 구합니다.

$$(s^2 - 1)Y(s) - s = \frac{1}{s+1}$$

$$(s^2 - 1)Y(s) = \frac{1}{s+1} + s = \frac{1 + s(s+1)}{s+1} = \frac{s^2 + s + 1}{s+1}$$

$$Y(s) = \frac{s^2 + s + 1}{(s+1)(s^2 - 1)} = \frac{s^2 + s + 1}{(s+1)^2(s - 1)}$$

#### 4단계: 부분분수 전개 (Partial Fraction Decomposition)

역변환을 쉽게 하기 위해 $Y(s)$를 부분분수 형태로 쪼갭니다.

$$Y(s) = \frac{A}{s-1} + \frac{B}{s+1} + \frac{C}{(s+1)^2}$$

계수를 구하면 다음과 같습니다.

- $A = \frac{3}{4}, \quad B = \frac{1}{4}, \quad C = -\frac{1}{2}$
    

#### 5단계: 역 라플라스 변환 (Inverse Laplace Transform)

변환표를 이용하여 시간 영역의 해 $y(t)$를 구합니다.

- $L^{-1}\{\frac{3/4}{s-1}\} = \frac{3}{4}e^t$
    
- $L^{-1}\{\frac{1/4}{s+1}\} = \frac{1}{4}e^{-t}$
    
- $L^{-1}\{\frac{-1/2}{(s+1)^2}\} = -\frac{1}{2} t e^{-t}$ (이동 정리 적용)
    

### 3. 최종 결과

주어진 초기값 문제의 최종 해는 다음과 같습니다.

$$\mathbf{y(t) = \frac{3}{4}e^t + \frac{1}{4}e^{-t} - \frac{1}{2} t e^{-t}}$$
