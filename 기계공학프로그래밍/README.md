# 기계공학프로그래밍

## 과목 개요

- **핵심 도구:** MATLAB
- **학습 목표:** 수학/공학 개념을 코드로 구현하고, 기계공학 문제를 계산적으로 해석하는 능력 향상
- **주요 내용:** 변수와 행렬 연산, 스크립트/함수 작성, 반복문·조건문, 그래프 시각화, 공학 응용 계산
