## 1. 기본 Line Plot (선 그래프)

### 1.1 기본 문법
```matlab
plot(xvalue, yvalue, '선색상 선스타일 마커', 'LineWidth', n)
xlabel('x축 이름과 단위')
ylabel('y축 이름과 단위')
title('그래프 제목')
grid on  % 격자 표시
```

### 1.2 주요 특징
- X와 Y 배열의 크기는 반드시 일치해야 함
- 선 색상, 스타일, 마커를 문자열로 지정
- LineWidth로 선 두께 조절

## 2. Subplot (여러 그래프 배치)

### 2.1 사용법
```matlab
subplot(1, 2, 1)  % 1행 2열 중 첫 번째 위치
% 첫 번째 그래프 그리기

subplot(1, 2, 2)  % 1행 2열 중 두 번째 위치
% 두 번째 그래프 그리기
```

- **형식**: `subplot(행, 열, 위치)`
- 하나의 Figure 창에 여러 독립적인 그래프 배치

## 3. Double-Axis Plot (이중 축 그래프)

### 3.1 개념
- 하나의 그래프에 두 개의 Y축 사용
- 값의 범위가 크게 다른 데이터를 함께 표시할 때 유용
- 예: 구리 가격(높음)과 철강 가격(낮음)을 동시에 표시

### 3.2 주요 함수
```matlab
yyaxis left   % 왼쪽 Y축 활성화
plot(...)
yyaxis right  % 오른쪽 Y축 활성화
plot(...)
```

### 3.3 hold on/off
- `hold on`: 현재 그래프를 유지하고 새 그래프 추가
- `hold off`: hold 상태 해제

## 4. Bar Plot (막대 그래프)

### 4.1 수직 막대 (기본)
```matlab
bar(x, y)
```

### 4.2 수평 막대
```matlab
barh(x, y)
```

### 4.3 카테고리 데이터 처리
```matlab
x = categorical({'항목1', '항목2', '항목3'});
x = reordercats(x, {'항목2', '항목1', '항목3'});  % 순서 변경
```

**중요**: `reordercats`는 **시각화 순서만** 변경하고, 데이터 값의 매칭은 변경하지 않음

### 4.4 색상 커스터마이징
```matlab
h = barh(x, y);
h.FaceColor = 'flat';
h.CData = [1 0 0; 0 1 0; 0 0 1; ...];  % RGB 값 (0~1)
```

- **사용자 정의 함수**를 만들어야 함

### 9.2 구조
```matlab

    % 사용자 정의 함수 구현
end
```

### 9.3 디버깅 팁
- 라인 번호 클릭 → 중단점(breakpoint) 설정

## 10. 고급 그래프 제어

### 10.1 객체 핸들 사용
```matlab
plot(x, y1)
hold on
plot(x, y2)
hold off

% 그래프 객체 가져오기
hkids = get(gca, 'Children');  % gca = get current axes

% 속성 변경
set(hkids(1), 'Marker', '*')
set(hkids(2), 'LineWidth', 4)
```

**주의**: Children은 **역순**으로 저장됨
- `hkids(1)` = 마지막에 그린 그래프
- `hkids(2)` = 첫 번째 그린 그래프

## 11. 애니메이션

### 11.1 기본 구조
```matlab
x = linspace(0, 2*pi, 1000);
h = plot(NaN, NaN, 'LineWidth', 2);  % 빈 플롯 생성
axis([0 2*pi -1.5 1.5]);

for t = 1:length(x)
    h.XData = x(1:t);
    h.YData = sin(x(1:t));
    drawnow;   % 즉시 그리기 (필수!)
    pause(0.01);  % 속도 조절
end
```

### 11.2 핵심 함수
- `drawnow`: 즉시 화면 업데이트
- `pause(초)`: 지정된 시간만큼 대기

## 12. 벡터장 시각화 (Vector Field)

### 12.1 수동 계산
```matlab
[x, y] = meshgrid(-2:0.2:2, -2:0.2:2);
V = x.^2 + y;
dx = 2*x;      % ∂V/∂x
dy = ones(size(x));  % ∂V/∂y = 1

contour(x, y, V), hold on
quiver(x, y, dx, dy), hold off
```

### 12.2 자동 계산
```matlab
[dx, dy] = gradient(V, 0.2, 0.2);  % 0.2는 격자 간격
quiver(x, y, dx, dy)
```

## 13. 3D 그래프 회전

```matlab
a = zeros(30, 30);
a(:, 15) = 0.2;
a(7, :) = 0.1;
a(15, 15) = 1;

el = 30;  % 고도각
for az = -37.5:15:-37.5+360
    mesh(a)
    view(az, el)  % 시점 변경
    pause(0.5)
end
```

- `view(azimuth, elevation)`
  - **azimuth**: 방위각 (수평 회전)
  - **elevation**: 고도각 (수직 각도)

## 14. 고품질 그래프 내보내기

### 14.1 GUI 방식
- Figure 창 → View → Plot Browser
- 속성 편집 (색상, 선 두께, 폰트 등)
- File → Save As → 형식 선택

### 14.2 프로그래밍 방식
```matlab
print('파일명', '-dpng', '-r300')   % 300 DPI PNG
print('파일명', '-depsc', '-r600')  % 600 DPI EPS (벡터)
print('파일명', '-dpdf', '-r300')   % PDF
```

### 14.3 해상도 권장사항
- 일반 문서: 300 DPI
- 학술 논문 (Nature 등): 600 DPI 이상
- 벡터 형식(EPS, PDF) 권장
