# 성균논어

성균논어 수업의 주차별 강의 내용과 핵심 논어 구문을 정리하는 저장소입니다. 배운 내용을 복습하고 과제를 준비하기 위한 자료들로 구성되어 있습니다.

## 📂 파일 및 폴더 구조

이 저장소는 주차별 강의 요약 파일과 논어 원문 해설 폴더로 나누어 관리됩니다.

```text
📦 성균논어
 ┣ 📜 README.md
 ┣ 📜 성균논어 4주차.md
 ┗ 📂 논어 구문, 해설
    ┗ 📜 4주차 논어 구문 해설.md
```

## 📝 상세 내용

### 1. 주차별 강의 노트 (`성균논어 X주차.md`)
 교수님의 주차별 강의 내용을 요약한 마크다운 파일입니다.
    
* 4주차: 예禮, 문화를 세우다
    

### 2. 논어 구문 및 해설 (`논어 구문, 해설/`)

- 강의 중 다루어진 『논어』 원문 구문만을 따로 추출하여 한자, 발음, 해석, 그리고 심층 의미를 분석한 문서들이 모여 있는 폴더입니다.
    
- 과제 작성 시 특정 구문을 인용하거나 상세한 뜻을 참고할 때 활용합니다.
