
# 열역학 

## 과목 소개

기계공학의 핵심 기초 학문으로, 에너지의 변환과 이동, 그리고 물질의 상태 변화를 다룹니다. 특히 검사 체적(Control Volume)을 중심으로 에너지 보존 법칙과 엔트로피를 학습하여 실제 공학 장치의 효율을 분석하는 것을 목표로 합니다.

## 학습 진도

- 1-2: 열역학의 기본 정의 (계, 주변, 검사 체적, 주요 가정)
    
-  2-1: 상태와 성질 (열역학적 평형, 강성적/종량적 성질, 과정과 사이클)
    
-  2-2: 순수 물질의 성질 (상변화 과정, T-v 도표, 건도)
    
-  3-1: 기체의 성질 (이상기체 상태 방정식, 압축률 계수, 물성표 해석)
	
*  3-2: 열과 일 (Heat & Work)
	
*  4-1: 닫힌 계에서의 열역학 제1법칙

## 메모

- **단위 환산 주의:** bar, Pa, atm 간의 환산 수치를 항상 확인할 것. ($1\text{ bar} = 10^5\text{ Pa}$)
    
- **상태 판단:** 문제를 풀 때 항상 포화 상태표(Saturation Table)를 먼저 확인하여 압축액, 포화 혼합물, 과열증기 중 어디에 해당하는지 판단할 것.
    
- **보간법 활용:** 표에 정확한 수치가 없을 경우 선형 보간법을 사용하여 중간값을 도출하며, 계산 실수에 유의할 것.
    
- **가정 확인:** 실제 공학 문제를 풀 때 연속체(Continuum) 및 정상 상태(Steady State) 가정이 유효한지 체크할 것.
