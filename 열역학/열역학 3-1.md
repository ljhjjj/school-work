
# 열역학 3-1: 순수 물질의 성질 (기체)

## 1. 기체의 기본 정의와 이상기체

- **유체 (Fluid):** 액체와 기체처럼 흐르는 성질을 가진 물질.
    
- **기체 (Gas):** 용기(Container)에 담았을 때 그 용기의 부피만큼 전체적으로 확산하여 채우는 특성이 있음.
    

### **이상기체 상태 방정식 (Ideal Gas Equation of State)**

보일의 법칙과 샤를의 법칙을 결합하여 유도된 식입니다.

$$Pv = RT \quad \text{or} \quad PV = mRT$$

- **기체 상수 ($R$):** $R = \frac{R_u}{M}$ (각 기체의 분자량 $M$에 따라 결정됨)
    
- **보편 기체 상수 ($R_u$):** 모든 기체에 공통으로 적용되는 값
    
    - $R_u = 8.314 \text{ kJ/kmol}\cdot\text{K}$
        
    - $R_u = \lim_{P \to 0} \frac{P \bar{v}}{T}$

## 2. 실제 기체와 압축률 계수 (Z)

실제 기체는 압력이 매우 낮을 때만 이상기체처럼 행동합니다. 이를 보정하기 위해 **압축률 계수($Z$**)를 도입합니다.
![Z vs P diagram for various temperatures](그림들/열역학/3-1-1%20Z_vs_P_diagram.png)
- **압축률 계수 정의:** $Z = \frac{P \bar{v}}{R_u T}$
    
- **그래프 해석:** * 압력이 $0$에 가까워지면($P \to 0$) 모든 기체는 $Z=1$인 이상기체와 같아집니다.
    
    - 온도가 낮아질수록 이상기체의 거동($Z=1$)에서 더 크게 벗어납니다.
        

### **일반화된 압축률 차트 (Generalized Compressibility Chart)**

다양한 기체들을 하나의 그래프로 분석하기 위해 **환산 상태량(Reduced Properties**)을 사용합니다.

![Generalized compressibility chart](그림들/열역학/3-1-2%20Compressibility_Chart.png)

- **환산 압력 ($P_R$):** $P_R = \frac{P}{P_{cr}}$
    
- **환산 온도 ($T_R$):** $T_R = \frac{T}{T_{cr}}$
    
- **사용법:** $P_R$과 $T_R$을 알면 차트에서 해당 지점의 $Z$값을 찾아 실제 기체의 부피나 압력을 구할 수 있습니다.

## 3. 열역학적 물성표 (Property Tables)

상태를 판단할 때는 **포화 상태표(Saturation Table**)를 가장 먼저 확인하는 것이 정석입니다.

### **상태 판단 기준 **

| **주어진 조건**     | **비교 결과**     | **상태 (Phase)**               |
| -------------- | ------------- | ---------------------------- |
| **온도($T$) 고정** | $P > P_{sat}$ | **압축액 (Compressed Liquid)**  |
|                | $P < P_{sat}$ | **과열증기 (Superheated Vapor)** |
| **압력($P$) 고정** | $T > T_{sat}$ | **과열증기 (Superheated Vapor)** |
|                | $T < T_{sat}$ | **압축액 (Compressed Liquid)**  |

### **표의 구성 및 특징**

1. **Saturation Table:** 온도가 일정하게 증가하는 표(Temperature table)와 압력이 일정하게 증가하는 표(Pressure table)가 있음.
    
2. **Superheated Vapor:** 온도와 압력이 모두 독립적인 변수이므로 두 값을 모두 알아야 물성치를 찾을 수 있음.
    
3. **Compressed Liquid:** 데이터가 많지 않으며, 표가 없을 경우 같은 온도의 **포화액($v_f, u_f, h_f$) 값**으로 근사하여 사용함.

## 4. 선형 보간법 (Linear Interpolation)

표에 내가 찾는 정확한 수치가 없을 때, 주변의 두 값을 이용해 직선 관계라고 가정하고 중간값을 구하는 방법입니다.

### **예시: $T=53^\circ C$일 때의 $P_{sat}$ 구하기**

- $T_1 = 50^\circ C \to P_1 = 12.350 \text{ kPa}$
    
- $T_2 = 55^\circ C \to P_2 = 15.758 \text{ kPa}$
    

**[계산 원리: 기울기가 일정하다는 성질 이용]**

$$\frac{P_{55} - P_{50}}{55 - 50} = \frac{P_{53} - P_{50}}{53 - 50}$$

> **Tip:** 시험에서는 교수님 성향에 따라 가장 가까운 값을 쓰라고 하실 수도 있지만, 정밀한 계산이 필요한 설계나 과제에서는 반드시 보간법을 써야 합니다.
