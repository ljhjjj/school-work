# 열역학 4-1: 닫힌 계에서의 열역학 제1법칙

## 1. 복습

- 에너지는 닫힌 계(Closed System)의 경계를 **열(Heat**)이나 **일(Work**)의 형태로 넘나들 수 있습니다.
    
- **에너지는 내부에 저장될 수 있지만, 열과 일은 저장될 수 없습니다.** (과정 중에만 나타남)
    
- **열 ($Q$):** 계와 주변 환경 사이의 온도 차이로 인해 경계를 넘어 전달되는 에너지.
    
- **일 ($W$):** 온도 차이 이외의 원인으로 전달되는 에너지. (물체를 들어 올리거나 가속하는 데 소모되는 에너지로 간주)
    

---

## 2. Path Independent vs. Path Dependent (경로 무관 vs 경로 의존)

본격적인 제1법칙을 다루기 전에, 열역학에서 가장 중요한 두 가지 변수의 성질을 구분해야 합니다.

- **Path Dependent (경로 의존, 경로 함수):**
    
    - 시작점(상태 1)과 끝점(상태 2)이 같더라도, **어떤 과정(경로)을 거쳤느냐에 따라 값이 달라지는 성질**입니다.
        
    - 대표적으로 **열($Q$**)과 **일($W$**)이 여기에 속합니다.
        
- **Path Independent (경로 무관, 상태 함수):**
    
    - 과정(경로)에 상관없이 **초기 상태와 최종 상태만으로 그 값이 결정되는 성질**입니다.
        
    - 온도($T$), 압력($P$), 부피($V$)와 같은 물성치뿐만 아니라, **에너지($E$), 내부 에너지($U$**)가 여기에 속합니다.
        

---

## 3. 단열 과정의 경우 (Adiabatic Process)

외부와 열 교환이 전혀 없는 상태($Q = 0$)를 단열 과정이라고 합니다.

- **단열 일의 경로 무관성 (Path Independent):** 일($W$)은 원래 경로 의존적(Path dependent)이지만, **단열 상태의 닫힌 계에서 두 평형 상태를 연결할 때 발생하는 일은 예외적으로 경로와 무관(Path independent)합니다.** 어떤 방식을 쓰든 오직 처음과 끝 상태에 의해서만 일의 양이 결정됩니다.
    
- 이 성질 때문에, 단열 과정에서의 일은 경로와 무관한 특정한 '상태 함수(에너지)'의 변화량으로 정의할 수 있게 됩니다.
    
- **단열 과정의 제1법칙 식:**
    
    $$-W_{adb, 1-2} = E_2 - E_1$$
    
- 운동 에너지($KE$)와 위치 에너지($PE$)의 변화량을 무시할 경우, 내부 에너지($U$)의 변화량으로 **정의**할 수 있습니다.
    
    $$-W_{adb, 1-2} \equiv U_2 - U_1$$
    

---

## 4. 비단열 과정의 경우

일반적으로 열 교환이 일어나는 상황입니다. 에너지는 열과 일의 형태로 동시에 출입할 수 있습니다.

$$-W_{1-2} + Q_{1-2} = U_2 - U_1$$

$$Q_{1-2} = \Delta U_{1-2} + W_{1-2}$$

- **부호 규약 (Sign Convention):**
    
    - $Q_{1-2} > 0$ : 계가 열을 **받음** (가열)
        
    - $Q_{1-2} < 0$ : 계가 열을 **방출함** (냉각)
        

---

## 5. 에너지 평형 (Energy Balance)

> **계의 에너지 변화 = 계로 들어온 열 전달량 - 계가 외부로 한 일**

전체 에너지 변화 기호($d$)를 사용하여 미분형으로 나타내면 다음과 같습니다.

$$d(U + KE + PE) = \delta Q - \delta W$$

$$dU = \delta Q - \delta W$$

(운동/위치 에너지를 무시할 경우)

이 식을 상태 1에서 2까지 적분하여 일반적인 에너지 평형 식을 완성합니다.

$$Q_{1-2} = (U_2 - U_1) + \frac{m}{2}(V_2^2 - V_1^2) + mg(z_2 - z_1) + W_{1-2}$$

$$Q_{1-2} = (U_2 - U_1) + W_{1-2}$$

(운동/위치 에너지 무시)

- **시간 변화율(Time Rate)로 표현:**
    
    $$\frac{dE}{dt} = \dot{Q} - \dot{W}$$
    

---

## 6. 사이클 과정 (Thermodynamic Cycle)

사이클은 상태 1에서 출발해 여러 과정을 거쳐 다시 상태 1로 돌아오는 것을 말합니다.

- 총 에너지 변화 수식 $\oint dE = \oint \delta Q - \oint \delta W$ 에서,
    
- 에너지($E$)는 **Path independent(경로 무관**)한 상태 함수이므로 제자리로 돌아오면 총변화량은 0이 됩니다. ($\oint dE = 0$)
    
- 즉, 사이클에서 총 열량은 총 일의 양과 같습니다.
    
    $$\oint \delta Q = \oint \delta W \implies Q_{net} = W_{net}$$
![](../그림들/열역학/4-1%20thermodynamic_cycle.jpg)

### 6-1. Work producing cycles (일을 하는 사이클)

- 열을 받아서 일을 생산해 내는 과정. (예: 자동차 엔진, 발전소)
    

### 6-2. Work consuming cycles (일을 받는 사이클)

- 일을 받아서 열을 이동시키는 과정. (예: 냉장고, 에어컨)
    

_(※ 제1법칙은 에너지가 보존된다는 것만 알려줄 뿐, 열이 스스로 고온에서 저온으로 흐른다는 방향성은 설명하지 못합니다. 이는 추후 **열역학 제2법칙**에서 다룹니다.)_

## 7. 엔탈피 (Enthalpy)

**정압 과정 (Constant pressure process, Isobaric process**)에서 열역학 제1법칙 식을 전개해 봅니다. 압력이 일정하므로 $P_1 = P_2 = P$입니다.

$$Q_{1-2} = U_2 - U_1 + W_{1-2}$$

$$Q_{1-2} = U_2 - U_1 + \int P dV$$

$$Q_{1-2} = U_2 - U_1 + P(V_2 - V_1)$$

$$Q_{1-2} = (U_2 + P_2V_2) - (U_1 + P_1V_1)$$

이때 계산의 편의를 위해 $U + PV$라는 새로운 물성치를 **엔탈피($H$)로 정의**합니다.

$$H \equiv U + PV$$

$$Q_{1-2} = H_2 - H_1$$

- **물리학적 의미:** 정압 과정(Isobaric process) 동안 계가 주고받은 **열 전달량**을 의미합니다. ($dH = \delta Q$)
    
- **참고 사항:** 등압 과정이 아닐지라도 $H \equiv U + PV$라는 정의 자체는 항상 성립합니다. 또한, 내부 에너지($U$)와 마찬가지로 단일 값 자체보다는 **변화량($\Delta H$**)이 공학적으로 훨씬 중요합니다.
    
- 포화 혼합물(Saturated liquid-vapor) 상태에서는 건도($x$)를 이용해 비엔탈피($h$)를 구할 수 있습니다. ($h = h_f + x h_{fg}$)
