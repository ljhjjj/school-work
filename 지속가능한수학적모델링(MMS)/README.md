# 지속가능한수학적모델링 (MMS)

Roe et al. (2018), *Mathematics for Sustainability*를 기반으로 지속 가능성(sustainability) 이슈를 수학적으로 분석하고 모델링하는 과목입니다.

## 과목 개요

- **핵심 주제:** 지속 가능성, 시스템 사고, 정량적 의사결정
- **학습 목표:** 환경·사회·경제 문제를 수학적 모델로 표현하고, 해석 결과를 통해 지속 가능한 선택을 평가
- **접근 방식:** 교재 이론 + PDF 정리 + 문제풀이를 통한 적용
